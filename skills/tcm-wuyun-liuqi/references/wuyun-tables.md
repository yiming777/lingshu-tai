## 一、干支推算

**公式**：
1. 年干 = (年份 - 4) % 10 → 0=甲, 1=乙, 2=丙, 3=丁, 4=戊, 5=己, 6=庚, 7=辛, 8=壬, 9=癸
2. 年支 = (年份 - 4) % 12 → 0=子, 1=丑, 2=寅, 3=卯, 4=辰, 5=巳, 6=午, 7=未, 8=申, 9=酉, 10=戌, 11=亥

**南北政**：甲己年为南政，其余为北政。

| 天干 | 化运 | 太过/不及 | 别名 |
|------|------|-----------|------|
| 甲 | 土运太过 | 太过 | 敦阜 |
| 乙 | 金运不及 | 不及 | 从革 |
| 丙 | 水运太过 | 太过 | 流衍 |
| 丁 | 木运不及 | 不及 | 委和 |
| 戊 | 火运太过 | 太过 | 赫曦 |

运性对应：土=湿，金=燥，水=寒，木=风，火=热。

| 次序 | 主运 | 平气名 | 时段 |
|------|------|--------|------|
| 初运 | 木（风） | 敷和 | 大寒起 73 日 |
| 二运 | 火（热） | 升明 |  |
| 三运 | 土（湿） | 备化 |  |
| 四运 | 金（燥） | 审平 |  |
| 终运 | 水（寒） | 静顺 |  |

### 客运排法

以当年岁运为初运，按 **木→火→土→金→水** 相生次序排五步。

例：甲年（土运太过）→ 初运土、二运金、三运水、四运木、终运火。

| 卯、酉 | 阳明燥金 | 少阴君火 |

### 客气六步排法

1. 三之气 = 司天
2. 按 **厥阴→少阴→太阴→少阳→阳明→太阳** 循环排列
3. 以司天为基点，前后各推两步，终之气 = 在泉

## 五、运气相合判定

| 类型 | 条件 | 病势评估 |
|------|------|---------|
| **天符** | 岁运与司天五行同 | 执法，其病速而危 |
| **岁会** | 岁运与年支五行同 | 行令，其病徐而持 |
| **同天符** | 太过之年 + 岁运与在泉同 | — |
| **同岁会** | 不及之年 + 岁运与在泉同 | — |
| **太乙天符** | 天符 + 岁会同一年 | 贵人，暴而死 |
| **平气** | 太过被司天抑制 / 不及得司天助 | 气化和平，发病较轻 |

## 六、六气客主加临分析

逐气分析客气与主气关系：

- **相得**：客主同气或五行相生 → 气候平稳
- **不相得**：客主相克
  - **主胜逆**：主气克客气 → 病重
  - **客胜从**：客气克主气 → 病较轻

## 七、三年化疫简述

《素问·刺法论》："天地迭移，三年化疫。"

刚柔失守（司天在泉之气不能正常迁正退位）→ 三年后可能引发疫病。

判断要点：上一年司天是否正常退位？当年司天是否正常迁正？→ 否 → 标记三年化疫风险。
