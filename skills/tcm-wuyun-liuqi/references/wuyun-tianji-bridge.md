## 一、为什么能融合

| | 五运六气 | 天纪 |
|---|---------|------|
| **核心问题** | 今年的气化偏向是什么？ | 这个人的命局偏向是什么？ |
| **时间尺度** | 年/季/节气 | 一生/大运/流年 |
| **空间尺度** | 天地大气 | 人之居所（阳宅） |
| **共享语言** | 干支、五行、阴阳 | 干支、五行、八卦、阴阳 |
| **与疾病关系** | 时病、疫病、外感 | 体质倾向、先天禀赋、命理疾病 |
| **灵枢台数据** | tcm-wuyun-liuqi | external/nihaisha/tianji.md |

**融合逻辑**：五运六气告诉你"今年为什么容易得这个病"，天纪告诉你"这个人为什么容易得这个病"。二者结合——**同一套干支系统，一个看天时，一个看人命。**

## 二、干支：共同的语言

### 两者共享的干支体系

五运六气和紫微斗数都建立在干支纪年之上。天纪中的命宫、四化、大运推算，和运气中的岁运、司天、客气推算，使用的是同一套天干地支。

| 干支维度 | 五运六气用法 | 天纪用法 |
|---------|------------|---------|
| **年干** | 化五运（甲己土/乙庚金...） | 定四化星（甲廉贞化禄...） |
| **年支** | 定司天在泉（子午少阴...） | 定命宫/身宫起法 |
| **五行** | 运性（风热湿燥寒） | 星曜五行（紫微属土...） |
| **阴阳** | 太过/不及 | 阳年/阴年 |

### 融合示例

以丙午年（2026）为例：

```
五运六气视角：
  丙=水运太过（流衍）+ 午=少阴君火司天
  → 全年水寒+火郁交织，寒热错杂，肾病/心病高发

天纪视角：
  丙=阳年，午=火
  → 丙年生人廉贞化忌，心脑血管/情绪问题需注意
  → 午年生人命宫午，太阳入命者午年需防心火过亢
```

两套结论不矛盾，互相印证——**寒热错杂**这个结论从两个角度都指向了同一方向。

## 三、五行/八卦的对应

### 五行→八卦→脏腑→运气

| 五行 | 八卦 | 脏腑 | 五运 | 六气 |
|------|------|------|------|------|
| 木 | 震、巽 | 肝/胆 | 风 | 厥阴风木 |
| 火 | 离 | 心/小肠 | 热 | 少阴君火/少阳相火 |
| 土 | 坤、艮 | 脾/胃 | 湿 | 太阴湿土 |
| 金 | 乾、兑 | 肺/大肠 | 燥 | 阳明燥金 |
| 水 | 坎 | 肾/膀胱 | 寒 | 太阳寒水 |

### 天纪阳宅方位 → 运气时空

天纪的阳宅理论中，不同方位（八卦位）影响不同脏腑。结合运气：

- **今年司天在泉对应方位** → 那些方位的房间/区域今年气化活跃
- **例：丙午年，少阴君火司天（离位/南方）** → 住宅南方区域今年火气偏盛，心火易亢
- **阳明燥金在泉（乾兑位/西北西）** → 下半年西北/西方区燥气偏盛，肺燥需防

## 四、临床应用路线

### 场景 1：辨证时加入命理维度

```
标准流程（现有）：
  四诊 → 五层漏斗 → 证型 + 经典出处

增强流程（加入天纪）：
  四诊 + 运气背景 + 患者出生干支（如有）
    → 五层漏斗
    → 证型 + 经典出处 + 命理倾向说明

输出增强：
  "基于当前丙午年水运太过+少阴君火司天的运气格局，
   与患者{出生年干支}的命理偏向叠加，
   其心肾系统今年承受的气化压力较常人更大。
   辨证结论中的{证型}与此时空格局高度吻合。"
```

### 场景 2：运气推算时加入方位建议

```
原输出：
  "今年岁运水运太过，寒湿偏盛，注意肾脾系统。"
  
增强输出：
  "今年岁运水运太过，寒湿偏盛。
   📐 天纪方位提示：住宅坎位（北方）今年水气活跃，
      若该位为卧室或常居处，肾系统负担可能加重。
      可考虑在坎位增火行元素（红色/暖色调）制衡。"
```

### 场景 3：疫病预测增强

五运六气本身有"三年化疫"理论。结合天纪的卦象周期：
- 运气推算"何时疫"
- 天纪推算"何地/何人易感"（阳宅方位+命理四化）

## 五、边界约束

### 五运六气的边界
- 纯内伤病不强行纳入运气
- 急诊/创伤不调用

### 天纪的边界（更严格）
- **天纪数据仅用于学术参考和命理学习**
- **不替代临床辨证，不作为独立诊断依据**
- **阳宅风水建议标注"传统术数参考，非居住硬性要求"**
- **紫微斗数命理分析标注"术数推演，仅供学习"**

### 两者叠加时的安全措辞

```
> 🌐 时空分析（五运六气 + 天纪）：
> 当前丙午年水运太过+少阴君火司天，
> 患者{命理信息}与此格局叠合。
> 此为天人相应的传统术数分析，不替代个体辨证。
> 辨证结论以四诊信息为准，术数推断仅作背景参考。
```
