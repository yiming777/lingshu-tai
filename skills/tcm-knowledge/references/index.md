## 加载策略

使用 `read` 工具带 `offset` 和 `limit` 参数精确定位，避免全量加载到上下文。

例如：
```
read(fangji-db.md, offset=100, limit=30)  # 只看温里剂部分
read(zhongyao-db.md, offset=200, limit=20) # 只看活血化瘀药部分
```
