## 一、根本命题

**《伤寒论》398 条不是方剂分类手册，而是邪气从体表到内里的「进军路线图」。** 每一经对应一条正经，每一证型对应这条经上的一个受阻位置。

传统模型把六经当作并列的"六大证候群类型"来分类；黎明素问模型把六经理解为**六条通路的水位观察口**——邪气走了哪条路、堵在哪一段、水的升降流动出了什么问题。

引用来源：`2022-10-03_《伤寒论》的中医解剖原理（绪论）.pdf`

### 六经辨证的讲解顺序（与《伤寒论》原文不同）

黎明素问提出：当今时代饮食寒凉，**太阴脾经寒湿已成为所有其他病症的根源**，六经的讲解辨证顺序应调整为：

| 顺序 | 经 | 理由 |
|---|---|---|
| **①** | **太阴脾经（核心）** | 饮食寒凉→99%亚健康人太阴脾经寒湿→根源 |
| **②** | 阳明胃经+厥阴肝经 | 太阴不升→阳明不降；太阴寒湿→厥阴郁结 |
| **③** | 少阴肾经+太阳膀胱经 | 脾阳虚→肾阳虚→膀胱经气不足→易外感 |
| **④** | 少阳胆经 | 现代人寒湿多→少阳热症已很少见 |

> ⚠️ 《伤寒论》按邪气从外向内顺序（太阳→阳明→少阳→太阴→少阴→厥阴），但临床辨证应先从太阴入手。

## 二、核心框架：一大循环 + 两条轴

### 人体大循环 = 水升火降（阴升阳降）

```
         ⬆ 少阳相火（热量）从顶向下敛降
         │
  太阳 ← ← 阳明 ← 少阳    ← 三阳（阳降：热量回收）
    ↓                        ↑
  体表散热 ← ← ← ← ← ← ← 热量向上扩散
    ↓                        ↑
  少阴 → 太阴 → 厥阴    → 三阴（阴升：水气升发）
         │
         ⬇ 肝木（水气）从底向上生发
```

### 关键差异：三阳不是"升"，三阳都在"降"

| 观点 | 传统教材常见表述 | 黎明素问 |
|---|---|---|
| 少阳 | 春升之气，疏泄条达 | 相火（热量）从顶向下敛降 |
| 太阳 | 人体最大阳经 | 体表散热通道——热量从内向外散发 |
| 阳明 | 多气多血 | 热量从外向下向内收敛 |
| 三阴 | 里虚寒证 | 水气（阴）从下向上蒸腾 |

**水升火降两个方向：**
- **水升（阴升）**: 少阴(水源) → 太阴(水运) → 厥阴(输布全身)
- **火降（阳降）**: 少阳(热量开始下降) → 太阳(体表发散) → 阳明(向内向下收敛归零)

> 引用来源：`2022-10-03_《伤寒论》的中医解剖原理（绪论）.pdf`、`2023-11-03_中医入门框架《六经辨证》总目录.pdf`

### 三组循环三上三下（黎明素问2023-11-18最终框架总结）

| 循环 | 足经1 | 手经 | 足经2 | 升降 | 经脉关系 |
|---|---|---|---|---|---|
| **第一组** | 太阴脾经 ⬆升 | 手太阴肺经 | ↔ | 阳明胃经 ⬇降 | 手阳明大肠经 | 脾(采购)·胃(销售)——物流循环 |
| **第二组** | 少阴肾经 ⬆升 | 手少阴心经 | ↔ | 太阳膀胱经 ⬇降 | 手太阳小肠经 | 肾(仓储)·膀胱(推广)——能量循环 |
| **第三组** | 厥阴肝经 ⬆升 | 手厥阴心包经 | ↔ | 少阳胆经 ⬇降 | 手少阳三焦经 | 肝(财务)·胆(产品)——核心循环 |

**三阴升 + 三阳降 = 一个完整人体大循环。** 脾不升→胃不降；肾不升→膀胱不降；肝不升→胆不降。

## 三、六经的经络解剖定位

### 三阳经——循行路线与病理对应

**1. 太阳病——足太阳膀胱经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于目内眦→上行巅顶→下项→挟脊→抵腰→走背→下至足 |
| 病理对应 | 头项强痛（后脑/头顶痛）、项背强几几、腰背痛、发热恶寒 |
| 核心病机 | 经气被寒邪束住——膀胱经不走气了→体表散热受阻→郁而发热 |
| 代表方剂 | 麻黄汤（寒闭——经气完全不通）、桂枝汤（风袭——经气部分不通） |

**2. 阳明病——足阳明胃经（+手阳明大肠经）**

| 项目 | 内容 |
|---|---|
| 循行 | 起于鼻旁→环唇→下颊→循喉咙→下膈→属胃→下行至足。**趺阳脉**在脚背高点，为胃气所系——病危可治与否的关键判断 |
| 两大生理特征 | **①血管极其丰富**——全身热迅速集中于此（脸部冬季不冷因阳明气血充盛）。阳明应凉降，若热不能回流→血液发热（糖尿病前期消渴症·白虎汤）
**②从上往下走**——食物从口入→胃消→小肠化→大肠出。通便是治疗感冒发烧/肺炎/肿瘤的重要要求 |
| 太阴阳明关系 | **脾经不升→胃经不降**，方向盘比喻：左边升不去→右边打不下来。太阴寒湿为根本时，通便不能用大黄（太阴寒凉），用枳实+厚朴；血虚血热便秘（女性多见）用凉血活血方根 |
| 腑实辨证 | 分阴证/阳证：阴证兼太阴寒湿→枳实厚朴；阳证（怕热汗出口渴苔黄脉数）→白虎/承气 |
| 代表方剂 | 白虎汤（气分热）、承气汤类（腑实热） |

**3. 少阳病——足少阳胆经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于目外眦→绕耳前后→下胸中→循胁→下行至足 |
| 病理对应 | 两侧头痛、口苦咽干目眩、胸胁苦满、寒热往来 |
| 核心病机 | 相火从顶向下敛降遇到障碍→热卡在胆经两侧 |
| 代表方剂 | 小柴胡汤（疏通胆经瘀热——柴胡+黄芩清上端，半夏+生姜降下端） |

### 三阴经——递进演变关系（与传统最大不同）

**黎明素问认为三阴病是递进演化的，不是并列的。**

```
太阴寒湿 ──恶化──→ 少阴寒+上热 ──恶化──→ 厥阴寒热交替
(吐利、腹满)      (脉微细、但欲寐、咽痛)     (消渴、气上撞心、厥热胜复)
```

**4. 太阴病——足太阴脾经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于足大趾→上行下肢内侧→入腹→属脾络胃→上行至舌（**底→中→顶三层**：底部在脚→中部腹部→顶部舌本舌下） |
| 底部症状 | 脚冷、小腿水肿、脚气、下肢冷痛、髋关节冷痛肥大——**寒凉下沉至脚** |
| 中部症状 | 腹部冷痛、大便粘马桶易溏稀、水桶腰、颈椎/肩周/腰椎病（腹部寒湿阻碍膀胱经气血） |
| 顶部症状 | 舌苔水滑白腻齿印·白痰泡沫·流口水·吐涎沫·眼袋晨起水肿·脸色灰暗/黄褐斑/酒糟鼻 |
| 核心病机 | 脾阳虚衰→寒湿内生→水气升发无力。太阴不升→阳明不降+肝经郁结 |
| 临床地位 | **太阴病是一切六经病的前提和基础。** 辨阴阳=先看舌苔确定是否有太阴寒湿→有即阴证→大量温药，凉药一味很小的量都不行！ |
| 代表方剂 | 理中汤（太阴虚寒吐利）、四逆汤（恶化为少阴时） |

**太阴病10大症状清单**（同时有2项以上即可下判断；基本症状项为最直接判断）：
| 类型 | 症状 |
|---|---|
| ⭐ **基本症状** | ①舌苔水滑·苔白/白腻/黄白/奶白/舌边齿印·舌中后部厚腻 |
| 辅助症状 | ②手脚易冷·腹部冷·大便粘马桶·易溏稀 |
| | ③白痰·痰中白泡沫多·易流口水·吐涎沫 |
| | ④失眠入睡困难·睡眠不深·厕所一滴水都听见 |
| | ⑤尿频·小腿水肿·脚气·下肢/髋关节冷痛 |
| | ⑥眼袋·晨起眼皮水肿 |
| | ⑦颈椎病·肩周炎·腰椎间盘突出 |
| | ⑧脸色灰暗·痘痘·黄褐斑·黑斑·酒糟鼻·面部凹陷/发泡 |
| | ⑨食欲差或食量大 |
| | ⑩说话声音小·力气小·易打嗝不停/喝水呛/跑步吐/晕车晕船 |

**太阴脾经三个循环影响：**
- 阳明胃经不降：失眠/长痘/打嗝/食量大/脸色暗——配合**半夏陈皮/桂枝泽泻/枳实厚朴**（轻症方根）
- 肝经郁结：腹痛胸闷/胸胁满——配合小建中汤方根（生姜炙草红枣饴糖+桂枝+白芍）
- 膀胱经气血受阻：颈椎/肩周/腰椎病症→减轻腹部寒湿即好转，甚至不用吃药

**5. 少阴病——足少阴肾经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于足小趾下→斜走足心→上行下肢内侧后缘→属肾络膀胱→循喉咙→挟舌本 |
| 病理对应 | 脉微细、但欲寐（精神萎靡）、咽痛（肾经走喉咙）、下利清谷、四肢厥逆 |
| 核心病机 | 太阴病恶化→肾阳亦衰→水寒于下+虚热浮于上（上热下寒） |
| 与其他经关系 | 少阴是**三阴病的中间阶段**——上承太阴（寒湿输布至此）、下启厥阴（寒化极则入厥阴） |
| 代表方剂 | 四逆汤（少阴虚寒）、白通汤（虚阳浮越）、黄连阿胶汤（阴虚火旺咽痛） |

**6. 厥阴病——足厥阴肝经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于足大趾上→上行下肢内侧→绕阴器→抵小腹→属肝络胆→上行至巅顶 |
| 病理对应 | 消渴、气上撞心（肝气冲心/胃——当归四逆汤证）、心中疼热、饥不欲食、厥热胜复 |
| 核心病机 | 最深层阴寒→阳气与寒邪在肝经交战→交替出现寒与热（寒热错杂） |
| 关键鉴别 | 厥阴病与少阴病的咽痛鉴别——少阴咽痛是肾经虚火上浮（尿清长、下肢冷、脉沉微）；厥阴咽痛是肝经瘀热上冲（小便黄、大便干、咽干口苦、脉弦数） |
| 代表方剂 | 乌梅丸（厥阴寒热错杂）、当归四逆汤（血虚寒凝+肝经顶端瘀热） |

> 引用来源：六篇核心 PDF（2022-10-03 六经 @ 太阳/阳明/少阳/太阴/少阴/厥阴）

### 六经→十二经络严格对应总结

| 经 | 足经 | 手经 | 升降 | 主要症状群（六经辨证判定依据） | 核心方根药 |
|---|---|---|---|---|---|
| **太阴** | 足太阴脾经 | 手太阴肺经 | ⬆升 | 舌苔水滑齿印+大便粘马桶+手脚冷+睡眠不深+眼袋+水肿+白痰泡沫+尿频——**全世界99%亚健康人都有** | 白术·茯苓·干姜·炙甘草·细辛·厚朴·花椒·吴茱萸 |
| **阳明** | 足阳明胃经 | 手阳明大肠经 | ⬇降 | 轻：呃逆打嗝+痰多+口臭+痘痘+水量呛（太阴不升故阳明不降）；重：白虎/承气（少见了当下） | 葛根(轻)·大黄/芒硝/石膏(重) |
| **少阴** | 足少阴肾经 | 手少阴心经 | ⬆升 | 阳虚下利不止+手足厥冷+身上恶寒+精神萎靡 | 附子(四逆汤) |
| **太阳** | 足太阳膀胱经 | 手太阳小肠经 | ⬇降 | 表寒：恶寒无汗(麻黄)/有汗恶风(桂枝)；水湿：五苓散(猪苓泽泻膀胱湿热) | 麻黄·桂枝·紫苏·猪苓·泽泻 |
| **厥阴** | 足厥阴肝经 | 手厥阴心包经 | ⬆升 | 重症：甲状腺肿大/乳腺肿瘤/子宫肌瘤/宫颈癌(肝经严重郁结)；轻：胸闷嗳气(四逆散)；上热下寒(乌梅丸)；血虚血热(齿疼/过敏/崩漏·六味方根) | 柴胡·白芍·枸杞·熟地黄·桃仁·红花 |
| **少阳** | 足少阳胆经 | 手少阳三焦经 | ⬇降 | 因太阴寒湿多见厥阴病，少阳热症较少。民国前太阴寒湿少则少阳热症多 | 柴胡·黄芩 |

**⚠️ 六经十二经络对应的临床意义：** 太阴脾经循行经过腹部→腹满/寒疝；肝经循行经过少腹乳腺甲状腺→三焦肿瘤；膀胱经循行背部→背冷恶寒。记住经脉巡行路线，不用背病名。

## 四、中药的"经络路径"重新理解

黎明素问对中药的理解不按"功效分类"走，而是按**"这味药作用于哪条经的哪个位置、帮助了哪个方向的循环"**来理解。

### 核心原则

每种药在体内有明确的**经络路径轨迹**——沿某条经的某个部位起作用，促进或抑制"水升火降"的某个环节。

### 代表药物重新理解

#### 柴胡
| 维度 | 内容 |
|---|---|
| 教材定义 | 疏散退热、疏肝解郁、升举阳气 |
| 黎明素问 | **通胆经上端瘀热、助胆火下降**——胆经从上向下走，第一道关口就是柴胡证（胸胁满、口苦）。柴胡不是"升"而是"通"——把胆经上端的热"顺"下去。 |
| 炮制观点 | 支持生用（醋酸制后效力大减） |
| 引用 | `2024-05-18_【柴胡】疏肝解郁.pdf` |

#### 白芍
| 维度 | 内容 |
|---|---|
| 教材定义 | 养血柔肝、平肝止痛、敛阴止汗 |
| 黎明素问 | **凉血柔胆、助胆火下降**——黄芩清胆经气分热（上端）、白芍清胆经血分热（助热从胆经下端向下走）。白芍与柴胡是胆经药对——柴胡通上端、白芍凉下端。 |

#### 虫药四味（虻虫·水蛭·蛴螬·蟅虫）
黎明素问对大黄蟅虫丸中四种虫药的祛瘀血机制做了生动的功能区分：
- **虻虫** → 锤子——打碎瘀血块
- **水蛭** → 小孩舔棒棒糖——把瘀血吸干净

### 方根理论的重要提示

**方根 ≠ 完整方剂。** 同一个方根可以出现在不同方剂中（如"生姜+大枣+炙甘草"同时出现在逍遥散和桂枝汤），但剂量和附加药物因主证而异。学习时应先拆解方根，再理解方剂，而非直接背诵完整方剂。

### 小建中汤方根拆解示范（2023-12-02太阴方根文章）

传统教材："小建中汤，温中补虚，和里缓急。主治中焦虚寒，肝脾不和证。"——初学者无法理解，更不会开方。

**方根拆解后一目了然：**

| 方根 | 药物 | 针对路径 | 经典原文关联 |
|---|---|---|---|
| 温太阴 | 生姜、炙甘草、红枣、饴糖 | 太阴脾经中焦寒凉 | 《金匮·虚劳》：虚劳里急，悸，衄，腹中痛，梦失精，四肢痠疼，手足烦热，咽干口燥→小建中汤主之 |
| 通经脉 | 桂枝 | 温通脾经肌腠经脉 | — |
| 疏厥阴 | 白芍 | 厥阴肝经血虚+郁结 | — |

**方根拆解为什么重要：** 歌诀（"小建中汤君饴糖，方含桂枝加芍汤"）无人能理解虚劳里急。但按六经拆解→立马清晰：患者同时有太阴寒(腹中痛) + 肝血虚(衄·梦失精+手足烦热·咽干口燥)。

### 方根拆解与剂量问答（实战）
问：常有方根里生姜和干姜的区别？
答：**生姜走而不守**（祛寒湿不走血分）vs **干姜守而不走**（温中守内）。有汗出怕热（厥阴阳明虚热）时换干姜。

问：逍遥散方根中为什么不用红枣（相比桂枝汤）？
答：逍遥散针对肝经郁结为主（柴胡+归+芍），红枣甘甜滋腻可能加重湿气。桂枝汤虚实夹杂需红枣护正。留给大家思考——这正是方根学习的精髓。

## 七、临床鉴别要点（从"药方错在哪"系列提炼）

黎明素问的4篇"药方错在哪"案例系列，展示了大量临床常见的辨证错误。以下是核心鉴别原则的提炼。

### 7.1 第一原则：先辨阴阳

这是黎明素问反复强调的根本原则：

> **只要出现了太阴证或少阴证，就是阴证。其他任何症状的热证，都是虚热、假热、假上火。**

**阴证判定（两者有其一即可）：**
| 经 | 核心指征 |
|---|---|
| 太阴证 | 舌苔水滑/白腻/白粉状、舌边齿印、食冷易溏、胃口差 |
| 少阴证 | 精神欠佳、困倦嗜睡、怕冷怕风、手脚冰凉、脉沉细微 |

**一旦判定为阴证：**
- 禁黄芩、黄连、黄柏、栀子、大黄、石膏等寒凉药（若必须用，≤1-2g且有温热药托底）
- 禁酸枣仁汤、小柴胡汤、五苓散等针对阳证的经方
- 禁用熟地、丹皮、丹参、生地黄等凉血药（血虚血热另论——区别见下）

引用来源：`2022-10-07_这张药方错在哪（耳鸣失眠怕冷症）.pdf`、`2022-10-09_这张药方错在哪（2）胃病心痛头痛症.pdf`

### 7.2 虚热 vs 实热的鉴别

| 维度 | 实热（阳证） | 虚热（阴证中的浮热） |
|---|---|---|
| 舌苔 | 舌红苔黄/苔厚，少津 | 舌淡嫩/水滑/白粉状为主，局部可薄黄 |
| 脉象 | 数而有力 | 沉、微、细为主，偶见微数 |
| 寒热 | 但热不寒 | 热中带寒——怕冷/手足凉/虽热但穿得多 |
| 大便 | 便秘/干燥 | 溏泻/软便/食冷即泻 |
| 小便 | 短赤/黄 | 清长或正常 |
| 口舌 | 口干喜冷饮 | 口干但不欲饮/即饮即泻 |

**黎明素问的关键观点：** 临床上90%以上的"上火"都是虚热——因为现代人多数有太阴寒湿（冷饮/牛奶/水果/空调所致）。

引用来源：`2022-10-07`、`2022-10-14`、`2022-10-21` 三篇药方错在哪

### 7.3 厥阴上热 vs 阳明实热的鉴别

这是黎明素问体系中最关键的鉴别之一：

| 维度 | 厥阴上热（虚热） | 阳明实热 |
|---|---|---|
| 舌苔 | 整体白/水滑，可薄黄 | 舌红苔黄燥 |
| 脉象 | 沉/弦，整体弱 | 洪大/滑数有力 |
| 寒热 | 底寒——怕冷/手脚凉/穿得多 | 但热不寒 |
| 大便 | 溏泻或正常 | 便秘燥结 |
| 口渴 | 口干但不欲饮或少饮 | 大渴引饮 |
| 胸胁 | 胸胁苦满+气上冲心 | 腹满硬痛 |
| 用药 | 柴+归+芍（疏肝养血清热） | 白虎/承气类（大清大下） |

**临床线索：** 患者有"上火"症状（口苦/咽干/胸闷/头痛/失眠）的同时，仍有怕冷/食冷易溏/苔白水滑/脉沉，则属于厥阴上热——不是阳明实热。

引用来源：`2022-10-07_这张药方错在哪（耳鸣失眠怕冷症）.pdf`、`2022-10-09_这张药方错在哪（2）胃病心痛头痛症.pdf`

### 7.4 失眠鉴别的三条路径

黎明素问强烈批评"失眠就用酸枣仁汤"的思维。失眠应按以下三条路径分别：

| 类型 | 核心指征 | 对应药对/方剂 |
|---|---|---|
| 血虚热心烦 | 舌红少苔、夜间燥热、心烦不卧 | 酸枣仁汤（炒枣仁+知母+川芎）——⚠️适用范围狭窄 |
| 心阳浮动惊狂 | 坐立不安、烦惊、心悸 | 龙骨+牡蛎（桂枝甘草龙骨牡蛎汤/柴胡加龙骨牡蛎汤） |
| 阴证水湿失眠 | 舌淡苔水滑、怕冷、食冷易溏、白天困倦、入睡困难/夜醒难再睡 | 不针对失眠用药，而治太阴少阴寒湿——湿去眠自安 |

**特别注意：** 第三种（阴证失眠）最为常见（现代人太阴寒湿体质），此时误用酸枣仁或龙骨牡蛎反而加重。

引用来源：`2022-10-07`、`2022-10-21`

### 7.5 药物使用的常见误区

**阿胶的狭窄场景：**
- 阿胶本质寒凉（非《本经》所述的"性平"，实际性寒）
- 仅适用于：出血明显的病症（咯血/便血/崩漏）+ 再生性贫血
- ⚠️ 绝大多数女性（太阴寒湿体质）不适合用阿胶——越吃身体越差
- 阿胶珠与阿胶无本质区别
- 引用：`2022-10-14_这张药方错在哪（3）皮肤斑疹例假提前症.pdf`

**酸枣仁的狭窄场景：**
- 仅适用于血虚热类型的失眠（舌红少苔+夜间燥热+心烦）
- ⚠️ 遇失眠就用酸枣仁汤，是抛弃六经辨证的最典型错误
- 引用：`2022-10-21_这张药方错在哪（4）夜醒失眠水肿症.pdf`

**黄芪的滥用警告：**
- 黄芪补的是太阴脾经中焦之气，不是"全身之气"
- 仅在明确的脾虚（苔白+乏力+纳差+脉弱）时可用
- ⚠️ 上焦有热（肺热/心火）或下焦有湿热时，黄芪"壅补"——把热锁在体内
- 引用：`2022-10-07`、`2022-10-14`、`2022-10-21` 三篇均批评黄芪滥用

**龙骨牡蛎不是安眠药：**
- 龙骨牡蛎治疗心阳浮动（坐立不安/烦惊/心悸），不是治疗"睡不着"
- 龙骨牡蛎寒凉——有三阴证者禁用
- 引用：`2022-10-07`

### 7.6 全面辨证的要求

黎明素问批评"一个完整经方对应一个病症"的思维——老年人/女性/慢性病患者通常是多经合病（最常见：太阴+少阴+厥阴三经同病），不能只用一个完整经方，必须根据各经邪气的比例灵活加减。

引用来源：`2022-10-21_这张药方错在哪（4）夜醒失眠水肿症.pdf`

### 7.7 斑 vs 疹的鉴别（血分 vs 气分）

这是黎明素问在阴阳毒中强调的重要鉴别：

| 类型 | 本质 | 表现 | 治法方向 |
|---|---|---|---|
| **斑** | 血热+瘀血——毛细血管已破裂，血入皮肤 | 手指按之不褪色，面赤斑斑如锦纹 | 凉血活血（升麻+当归+鳖甲） |
| **疹** | 邪气欲出不能出——毛细血管末端充血，未破 | 手指按之褪色 | 疏通出路（麻黄透表/柴胡透少阳/葛根透阳明） |

**关键鉴别：** 斑在血分（肝经），疹在气分（太阳+少阳+阳明）。**升麻鳖甲汤治斑不治疹。**

引用来源：`2022-11-09_金匮3-3阴阳毒的治法.pdf`

### 7.8 凉血活血必须反佐热药（金匮虚劳治法的核心原则）

从金匮虚劳篇（6-1至6-3）所有核心方剂中提炼的一条共同原则：

> **大量凉血活血药之中，必须用小剂量温热药反佐，否则血热未清而阳气已伤。**

| 方剂 | 凉血药 | 反佐热药 | 反佐机制 |
|---|---|---|---|
| 八味肾气丸 | 地黄+丹皮+泽泻（大量凉血） | 桂枝1+附子1（极少量） | 启阳引阴——热药引凉药入少阴，附子起阳以生阴 |
| 酸枣仁汤 | 酸枣仁+知母（收敛血热） | 川芎2（小量） | 防止酸枣仁收敛过度——留一丝血运通道 |
| 炙甘草汤 | 生地黄16+麦冬+麻仁+阿胶（大量滋阴） | 桂枝3 | 温通血脉——免血脉过凉而瘀滞 |
| 薯蓣丸 | 柴胡+当归+熟地+白芍+川芎+阿胶（疏肝凉血） | 桂枝10 | 平衡大量凉血活血药的寒性 |
| 大黄蟅虫丸 | 大黄+黄芩+熟地+白芍（寒凉） | 干漆+杏仁（平性） | 丸剂缓治，小量渐进 |

**临床启示：** 任何厥阴病（肝经郁结+血虚血热）的方剂中，必须反佐小量温热药（桂枝/川芎/小量附子/干姜），这是金匮虚劳篇教给我们的核心组方法度。

引用来源：`2022-12-11_金匮6-3虚劳的治法_阴虚.pdf`

### 7.9 时疫（新冠）之阴证鉴别

2022年12月第一轮新冠中，黎明素问总结了一条极其重要的判断标准：

> **多数感染者舌苔湿滑（太阴寒湿），本质是阴证，不是阳证。** 凡是不口渴、舌苔灰白湿滑腻、不是苔黄、不是汗出怕热——**绝对不能使用含寒凉药物的方剂（连翘/黄芩/金银花/薄荷/石膏/大黄等）**。

| 维度 | 正确思路 | 错误思路（当时主流） |
|---|---|---|
| 证型识别 | 太阴寒湿底+太阳膀胱经被封闭+邪气湿为主 | 误判为热毒/风热/上火 |
| 核心症状 | 喉痛（假热）+腰痛（太阳经脉）+高烧身烫（闭热）+舌苔水滑 | 只看喉痛高烧→断为热证 |
| 核心方剂 | 人参败毒散（宋代·《和剂局方》·明清多次瘟疫验证） | 连花清瘟等清热中成药 |
| 药后表现 | 1天退烧+舌苔立即改善+咳嗽大减 | 寒凉药加重太阴寒湿→邪气入里→咳嗽迁延 |
| 后遗症本质 | **没有后遗症。** 所谓的后遗症=邪气未净。分经论治（太阳少阳→柴胡桂枝汤；阳明→大柴胡汤；太阴→理中四逆类），邪气出尽则愈。 | 后遗症当作独立病种治疗→反复用寒凉药→邪气愈压愈深 |

**人参败毒散的核心药组：**
- 羌活+独活+柴胡+前胡+川芎：解太阳+少阳+厥阴表湿
- 枳壳+桔梗+茯苓+甘草：太阴运湿
- 人参：扶正气以外托（非补虚——为托邪外出）

引用来源：`2022-12-15`、`2022-12-21`、`2022-12-26` 三篇新冠系列

### 7.10 肺痿 vs 肺痈（阳虚水湿 vs 阴虚血热）鉴别

| 维度 | 肺痿（阳虚咳喘） | 肺痈（阴虚咳喘） |
|---|---|---|
| 本质 | 阳虚水湿为主+轻度瘀热（阴证虚热） | 阴虚血热+血瘀化脓（阳证实热） |
| 脉象 | **虚数**——不耐按，反弹力不足 | **滑数**——反弹有力，滑而数 |
| 痰饮 | 白痰/稀痰/部分浓痰——津液煎熬 | 脓血腥臭痰——血热瘀血化脓 |
| 症状 | 口干不渴（总体阳虚水湿） | 咽干口燥 |
| 治法 | 温热祛寒+祛湿燥痰 | 清热凉血+排脓化瘀 |
| 代表方 | 甘草干姜汤（肺冷）/苓桂术甘汤（痰饮） | 麦门冬汤（肺热）/桔梗汤（排脓）/苇茎汤（清透瘀热） |
| ⚠️ 误区 | **川贝炖梨仅适用于肺痈（黄苔口干渴胸热汗出的阳证）**，多数人（阳虚水湿）越吃越严重 | — |

引用来源：`2023-01-03`、`2023-01-14` 两篇肺痿肺痈

### 7.11 腹满虚实鉴别

黎明素问强调：**腹满辨虚实，是决定用温药还是下法的第一关。**

| 维度 | 虚证（寒） | 实证（热） |
|---|---|---|
| 触诊 | **按之不痛**——按下去更舒服（蹲下缓解） | **按之痛**——拒按 |
| 时间 | **发作有时**——时好时坏（正气推寒湿） | **持续不减**——不减或减不足言 |
| 舌苔 | 白滑/水滑/白粉状 | 苔黄燥/黄厚 |
| 脉象 | 弦紧/沉细/微 | 滑数/大/沉实有力 |
| 治法 | **温药**——附子粳米汤/大建中汤/大黄附子汤（温下） | **下法**——厚朴三物汤/大承气汤 |

**特殊：大黄附子汤**——胁下偏痛+触痛+发热（局部热）+整体脉紧弦（整体寒）。这叫"温药下之"——大热药（附子+细辛）+大黄。这是温下法的代表，与阳明实热的大承气汤完全相反。

引用来源：`2023-02-06`、`2023-02-12` 两篇腹满寒疝宿食

### 7.12 四饮鉴别（水邪分布位置）

黎明素问将饮邪按位置分为四类：

| 类型 | 位置 | 表现 | 代表方 |
|---|---|---|---|
| **痰饮（淡饮）** | 心下/胃/脾/脐下 | 肠间沥沥有声+素盛今瘦+短气+目眩/冒眩 | 苓桂术甘汤（水湿初起·无热）/泽泻汤（湿久热郁·重冒眩） |
| **悬饮** | 胁下一侧 | 咳唾引胁下痛（类似现代胸腔积液） | 十枣汤 |
| **溢饮** | 四肢/皮肤/肌肉层 | 水湿满溢四肢——身体疼重 | 大青龙汤（无寒+烦躁）/小青龙汤（心下有水+寒） |
| **支饮** | 心肺两脏之间 | 咳逆倚息+短气不得卧+其形如肿 | 葶苈大枣泻肺汤/小青龙汤/木防己汤 |

**关键鉴别：** 四种饮的本质相同（阳虚水湿），只是水道阻滞的位置不同。治疗的总原则一致：温阳化饮（苓桂术甘汤是化痰饮的基础方）。不同点在于加味方向——表者汗之（大小青龙）、下者利之（苓桂术甘/泽泻/甘遂）、中者温之（苓桂术甘/甘遂半夏）。

### 7.13 消渴六经分类（类"糖尿病"的辨证鉴别）

黎明素问批评"消渴=糖尿病"的对号入座思维——消渴按六经途径分为以下四种：

| 类型 | 定位 | 核心症状 | 代表方 |
|---|---|---|---|
| **太阳水循环瘀堵** | 膀胱出口受阻 | 脉浮+小便不利+微热+消渴+水入则吐 | 五苓散 |
| **阳明经热津伤** | 阳明胃热+气盛 | 脉洪数+怕热汗出+口干舌燥+大便坚+小便数+消谷引食 | 白虎加人参汤 |
| **肾阴虚火旺** | 肾阴亏损+虚火上炎 | 寸口脉浮迟+口干不解渴（"渴欲饮水不止"）+无热象 | 文蛤散（咸寒清肾中虚火） |
| **厥阴寒热错杂** | 手厥阴心包经 | 消渴+气上冲心+心中疼热+饥而不欲食+食即吐——⚠️ 很多人误当厥阴病主方，实际仅属于手厥阴心包经的一个类型 | 乌梅丸 |

**男子消渴+饮一斗小便一斗**=阴阳两虚+尿失禁（非三多糖尿病）→肾气丸。

⚠️ **核心提醒：** 跟西医的对号入座"糖尿病"不一样——怕冷怕热不一样、大便干燥湿粘不一样、胃口饭量不一样，治法完全不同。西医"降血糖"思路与中医"调六经"思路本质不同。

### 7.14 痰饮辨证案例反思（小青龙汤误治的经典教学）

黎明素问从《金匮》12-4的经典辨证案例中提炼了极其重要的临床原则：

> **第一步：不可凭3个症状就套一个经方。** 首剂"咳逆倚息不得卧"→小青龙汤主之——但这个判断过于武断，忽略了患者整体的阴血+阳气+水湿分布。

> **第二步：辨证须把六经（12经）所有病症都罗列出来。** 分别归类到不同的经上去，对应出方根，再成方剂。不是用完整经方去套。

> **第三步：1-2剂一调的节奏才是仲景本意。** 一次性开7-14剂(尤其慢性病)，吃完错到哪里去都不知道，坏病都吃出来了。

**案例的精炼调整路径：**
| 步骤 | 方剂 | 为什么调 |
|---|---|---|
| ① 首诊 | 小青龙汤（原方套用） | 片面——只有"咳逆倚息不得卧"3个症状 |
| ② 变证 | 桂苓五味甘草汤（茯苓+桂枝+甘草+五味） | 服小青龙后冲气上逆+虚阳上浮+面热如醉 |
| ③ 变形 | 苓甘五味姜辛汤（去桂+干姜+细辛） | 冲气已平+咳喘仍盛+胸满 |
| ④ +半夏 | +半夏（苓甘五味姜辛夏汤） | 呕吐+冒眩（心下水饮未净） |
| ⑤ +杏仁 | +杏仁（再加杏仁汤） | 形肿（肺气阻滞+气血虚不能用麻黄） |
| ⑥ +大黄 | +大黄（再加大黄汤） | 面热如醉+胃热上冲→以利之 |

**关键教训：** 患者本质为厥阴肝经气郁内热+血虚+阳明热证+太阴水湿。小青龙温燥发散不适合这种基本盘。正确的辨证须全面收集六经信息，而不是凭三四个症状就套经方。

### 7.15 五水鉴别（风水·皮水·正水·石水·黄汗）

黎明素问认为：水气与湿气、痰饮本质相同（水湿），只是程度不同——湿气最轻→痰饮次之→水气最重。

| 类型 | 本质 | 脉象 | 核心症状 | 发汗/利水 | 代表方 |
|---|---|---|---|---|---|
| **风水** | 风+水——太阳表风+水湿溢出 | 浮 | 恶风+骨节疼+身痒+面目肿+洪肿+汗出乃愈 | **发汗** | 防己黄芪汤（汗出恶风+脉浮身重·表虚）vs 越婢汤（一身悉肿+无大热+脉浮不渴·表实有热） |
| **皮水** | 全部水饮溢出——无风邪 | 亦浮 | **不恶风**（关键鉴别）+胕肿按之没指+腹如鼓+不渴+四肢聂聂动 | **发汗** | 防己茯苓汤（四肢肿+聂聂动）/ 越婢加术汤（有热）/ 甘草麻黄汤（无热）/ 杏子汤（脉浮柔和） |
| **正水** | 太阴脾经寒湿+水停胸腹 | 沉迟 | 腹满+**喘**（与石水关键区别）+咳 | **利小便** | 枳术汤（心下坚如盘+气滞水湿）/ 小青龙（心下有水+风寒+饮）/ 小半夏加茯苓（呕+眩悸） |
| **石水** | 肾阳亏损+水结少腹 | 自沉 | 腹满+**不喘**（与正水关键区别）+脐肿+不得溺+足逆冷+面反瘦 | **利小便** | 肾气丸/四逆类温阳利水 |
| **黄汗** | 脾阳虚+阴血瘀热+汗出入水 | 沉迟 | 汗沾衣色黄如药汁+身发热+胸满+四肢头面肿+**两胫自冷**（与历节鉴别） | √ | 黄芪芍桂苦酒汤（大量黄芪+芍药苦酒凉血+桂枝祛水湿）/ 桂枝加黄芪汤（上身汗下身无汗+小便不利） |

**腰以上肿→发汗乃愈；腰以下肿→利小便。** 此金匮水气病总则。

**核心鉴别矩阵：**
| 问题 | 鉴别 |
|---|---|
| 有无风邪在表？ | 恶风=风水 vs 不恶风=皮水/正水/石水/黄汗 |
| 有没有喘？ | 喘=风水/正水 vs 不喘=石水/皮水/黄汗 |
| 汗出颜色？ | 色黄如药汁=黄汗 |
| 两胫冷热？ | 两胫冷=黄汗 vs 两足热=历节 |
| 表实vs表虚？ | 汗出恶风→防己黄芪汤(表虚) vs 无汗有热→越婢(表实) vs 无汗无热→甘草麻黄汤(表实) |

引用来源：`2023-04-02`、`2023-04-09`、`2023-04-16`、`2023-04-23` 四篇水气病

### 7.16 五脏水鉴别（心水·肺水·脾水·肝水·肾水）

五脏水是水气病在皮水和正水之间的交叉类型——水邪侵入某一脏：

| 类型 | 核心症状 | 病机 |
|---|---|---|
| **心水** | 身重+少气+不得卧+烦而躁+阴肿 | 寒水凌心→心阳耗散+心火上炎烦躁+下焦肾阳不足水泛 |
| **肺水** | 身肿+小便难+**时时鸭溏** | 水邪迫肺→肺不通调水道+浸淫大肠→鸭溏 |
| **脾水** | 其腹大+四肢苦重+少气+小便难 | 湿困脾胃→不能升清降浊+水湿聚于中下焦 |

| **肾水** | 腹大+脐肿+腰痛+不得溺+阴下湿+足逆冷+面反瘦 | 肾阳衰弱→不能温化水饮+不能蒸腾向上→上焦不足面瘦 |

### 7.18 黄疸三型鉴别（谷疸·女劳疸·酒疸）

|---|---|---|---|
| **谷疸** | 风寒相搏+食谷不化+湿热瘀堵 | 寒热不食+食即头眩+心胸不安+小便不通→发黄 | 小便不利（湿热无法排出）+脉紧迟 |

| 类型 | 来源 | 辨证要点 | 治法 |
|---|---|---|---|
| **经水适断+如疟状** | 经期外感→邪入少阳+肝经胆经气机郁结 | 续来寒热+发作有时+经水中断 | **小柴胡汤**（疏肝清胆+姜枣参草温中） |
| **经水适来+暮谵语** | 经期伤寒发热+热入血室 | 昼日明了+暮则谵语如见鬼状 | **小柴胡汤**（⚠️无犯胃气及上二焦） |
| **经水适来+热除脉迟+胸胁满如结胸** | 中风入血室日久·瘀热较重 | 身凉和+脉迟+胸胁满如结胸状+谵语 | **刺期门**（放肝经瘀热）+小柴胡汤 |
| **阳明病下血谵语** | 非经期·阳明邪热入胞宫迫血 | 但头汗出+身上无汗（厥阴肝经不通） | **刺期门**（泻之·濈然汗出愈） |

**核心原则：** 热入血室=经期胞宫空虚+邪气内陷。阳证属少阳小柴胡，病久瘀热重刺期门。

引用来源：`2023-09-03_金匮22-1妇人杂病.pdf`

## 八、《金匮要略》的六经路径视角

黎明素问认为《金匮要略》的所有病证都必须回到六经辨证框架来理解，不能按病名生搬硬套。以下是金匮核心病证的六经定位。

### 8.1 痉病（抽搐）

| 维度 | 内容 |
|---|---|
| 本质 | **肌肉津液匮乏的热证**——三阳证，不是三阴证 |
| 太阳柔痉 | 太阳病+汗出+脉沉迟→栝蒌桂枝汤（天花粉生津润燥） |
| 太阳刚痉 | 太阳病+无汗+口噤+气上冲胸→葛根汤（葛根透肌肉深处邪气） |
| 阳明刚痉 | 胸满+口噤+卧不着席+脚挛急+齘齿→大承气汤（清阳明腑实救津液） |
| 少阳痉 | 少阳病+胸胁苦满+抽搐→小柴胡汤+生石膏 |
| ⚠️ 三阴无痉 | 三阴证（太阴寒湿+少阴阳虚+厥阴血虚）**不出痉病**，只有抽筋/手抖/抖腿——肝经血虚风动，与痉病的肌肉津液枯竭不同 |

引用来源：`2022-10-15_金匮2-1痉病（抽搐）的治法.pdf`

### 8.2 湿气病

| 维度 | 内容 |
|---|---|
| 本质 | 水湿瘀堵在体表或肌肉层 |
| 阴黄（太阴） | 湿重于热，颜色暗如"熏黄"→茵陈五苓散（祛湿为主） |
| 阳黄（阳明） | 热重于湿，颜色鲜艳如"橘黄"→茵陈蒿汤（清热为主+祛湿） |
| 麻黄加术汤 | 湿在表——以身烦疼为主+脉浮+苔白滑 |
| 麻杏薏甘汤 | 湿+瘀热在表——以身疼+下午（日晡）发热加重为主 |
| 汗法要点 | **微汗出**——大汗则风去湿留 |
| 湿在头 | 寒湿在头→瓜蒂散吹鼻（擤出黄水） |

### 8.3 中暑（中暍）

| 类型 | 本质 | 核心症状 | 治法 |
|---|---|---|---|
| 暑在火中 | 阴虚火旺体质+暑热 | 汗出+身热+口渴（白虎加人参汤证） | 白虎加人参汤——清热生津 |
| 暑在湿中 | 阳虚水湿体质+暑热 | 身热+疼重+脉微弱（一物瓜蒂汤证） | 一物瓜蒂汤——祛湿则暑无所依附 |

### 8.4 百合病（精神恍惚）

| 维度 | 内容 |
|---|---|
| 本质 | **阴虚内热→心血不足→精神恍惚** |
| 与心阳浮动鉴别 | 百合病（心血不足）vs 龙骨牡蛎证（心阳浮动）——前者安静恍惚，后者坐立不安烦惊 |
| 核心指征 | 口苦+小便赤+脉微数（其他症状不一：想吃不想吃，想睡睡不着，如寒无寒如热无热） |
| 主方 | 百合地黄汤（百合清心凉血+生地黄活血祛瘀） |
| 变方 | 百合知母汤（误汗后）、滑石代赭汤（误下后）、百合鸡子汤（误吐后） |
| ⚠️ 注意 | 百合病的热在血分（无实热），不可用大寒大下之法 |

### 8.5 金匮学习的根本原则

> **《金匮》的病名≠病因。** 用病名去套方剂是最大误区——面对一个病人，还是要回到六经辨证上来：六经上分别有哪些邪气，分别用哪些药物，这个思维模型不能少。

### 8.6 狐惑病（类手足口病/口腔溃疡）

| 维度 | 内容 |
|---|---|
| 本质 | **脾胃湿热→湿热上攻/下潜→关口溃烂** |
| 上蚀咽喉（惑） | 口腔/舌头/咽喉溃烂→甘草泻心汤（炙甘草解毒+芩连清热燥湿+姜夏行气燥湿+参枣补脾） |
| 下蚀前阴（狐） | 前阴溃烂瘙痒→苦参汤外洗 |
| 蚀于肛 | 肛门溃烂→雄黄烧烟熏之 |
| 脓已成 | 目四眦黑（脓血到了眼睛）→赤小豆当归散（赤小豆发芽排脓） |
| ⚠️ 鉴别 | 甘草泻心汤类方的区别：半夏泻心汤·心下痞（燥湿祛痰）vs 生姜泻心汤·水湿肠鸣（温阳）vs 甘草泻心汤·滋阴清热（狐惑专方）vs 附子泻心汤·浮热+阳虚（汗出恶寒） |
| 现代应用 | 口腔溃疡本质=脾虚（下）+胃热（上）。上热芩连、下寒姜夏草参枣，比例调整。口腔溃疡反复发作者，多属此证。 |

引用来源：`2022-11-04_金匮3-2狐惑病的治法.pdf`

### 8.7 阴阳毒（传染病/血热入血）

| 维度 | 内容 |
|---|---|
| 本质 | **疫气直入血分→血脉凝结**（非单纯杀菌/杀病毒思路） |
| 阳毒 | 面赤斑斑如锦纹（血热瘀血）+咽喉痛+唾脓血→升麻鳖甲汤（升麻托毒外出+当归鳖甲活血+蜀椒雄黄燥湿杀虫） |
| 阴毒 | 面目青（厥阴寒）+身痛如被杖（血瘀）+咽喉痛→升麻鳖甲汤去雄黄蜀椒（阴证阳气已衰，不必燥湿杀虫） |
| 核心药理 | 升麻入太阴→托毒从阳明而出（升麻不升阳，而升脾运脾以托毒） |
| 斑vs疹鉴别 | 斑（血分→按之不褪色·升麻鳖甲汤）vs 疹（气分→按之褪色·麻黄/柴胡/葛根透表） |
| 凶险程度 | 5日可治，7日不可治（大概数字）——直入心肝二脏，6-7天可致命 |
| 现代应用 | 猩红热、红斑狼疮、紫癜病、血小板减少性紫癜属热毒血瘀者 |

引用来源：`2022-11-09_金匮3-3阴阳毒的治法.pdf`

### 8.8 疟病

| 维度 | 内容 |
|---|---|
| 本质 | **邪犯少阳为主**（疟脉自弦），非单纯杀疟原虫 |
| 五型鉴别 | ①瘅疟（热疟）：但热不寒→白虎汤/竹叶石膏汤 |
|  | ②温疟：热多寒少+骨节疼烦（内热外寒）→白虎加桂枝汤 |
|  | ③牡疟：寒多热少+心阳被寒痰包裹→蜀漆散（蜀漆+云母+龙骨）、牡蛎汤（兼风寒） |
|  | ④劳疟：疟久+正气虚+口渴→柴胡去半夏加瓜蒌汤 |
|  | ⑤疟母：疟久成症瘕→鳖甲煎丸 |
| 关键原则 | 按六经辨证加减，非照搬方剂。服药在疟疾发作前2小时为佳。 |
| ⚠️ 青蒿素 | 青蒿素提取已不属于中医治法（杀虫思路），中医思路是驱邪外出——永远不会产生耐药性 |

### 8.9 症瘕积聚（类肿瘤）

| 维度 | 内容 |
|---|---|
| 本质 | **痰+血的结合体**——疟邪久踞少阳→深伏经隧→与气血痰湿聚结 |
| 主方 | 鳖甲煎丸（23味）——但需按六经辨证加减，不可照搬 |
| 主思路 | 柴胡桂枝干姜汤（疏通少阳气机，领邪外出）+ 鳖甲灶灰清酒（软坚散结核心制法）+ 丹桃草药（逐瘀血化痰下水）+ 虫四味（蜂巢蟅虫蜣螂鼠妇→搜剔络中伏邪） |
| ⚠️ 安全 | 肿瘤患者通常已病重：①未手术化疗者方剂尚可 ②已手术者半条命 ③已化疗（尤其脱发）者阳气极衰。**必须先立二本（太阴脾少阴肾），再逐步用药**，不可直接用大量虫药/寒凉药。 |
| 关键是 | 全身不止一个大肿瘤——小病灶早就遍布全身。虫药搜邪+活血化瘀把全身处理干净，而非手术切除一地又长一地。 |

### 8.10 中风（狭义+广义）

| 维度 | 内容 |
|---|---|
| 本质 | **血虚血热→肝风内动→痰阻经络**（不是单纯的"血管堵了"） |
| 中风四阶段 | 邪在于络（肌肤不仁→麻木）→邪在于经（重不胜→肢体重滞）→邪入于腑（不识人→湿浊上攻头）→邪入于脏（舌难言口吐涎→心肝受邪） |
| 狭义中风（卒中） | 续命汤系列——少阴用附子+太阴用姜草参+厥阴血热用归芍芎+太阳风寒用麻桂杏（或防己防风）+夹痰用半细心 |
| 广义中风（夹寒） | 侯氏黑散（菊花防风驱风+黄芩清少阳+明矾牡蛎桔梗祛痰+归芎温血+术苓姜参温中+桂细心温经）——寒热矛盾，需慢调（酒服方寸匕，日一服） |
| 广义中风（火热） | 风引汤（五脏火热内盛+血压飙升+面红口苦便秘）——桂枝甘草龙骨牡蛎汤+大黄干姜+寒水石滑石石膏紫石英等 |
| ⚠️ 牛黄丸滥用 | 仅适用于热痰迷心包的中风重症。轻痰（热痰）用菖蒲远志制胆星竹沥替代。阳虚水湿体质的中风（寒痰）禁用。 |
| **预防＞治疗** | 中风前20-30年就有早期信号（眼干眼花·手麻抽筋·夜醒失眠·皮肤斑疹·胸闷短气·腰酸背痛）——这些是厥阴病血虚血热早期表现，早期治疗可完全杜绝中风。长期吃降压药而忽略阳虚水湿+血虚血热两大内因，才是中风的真正推手。 |

**续命汤 vs 三黄汤 vs 术附汤（三种中风证型的鉴别）：**

黎明素问在金匮5-4中对此做了精炼总结：

| 类型 | 本质 | 症状特点 | 方剂 |
|---|---|---|---|
| 续命汤证 | 体内阴阳两虚（阳气+阴血不足）+风邪侵入 | 半身不遂+口眼歪斜+痰涎 | 续命汤（麻桂姜归芎+补阳+补血化痰+人参） |
| 三黄汤证 | 表虚不固+风热湿邪侵入（无阳虚阴虚底） | 手足拘急+百节疼痛+恶寒+烦热心乱不思食 | 麻黄+独活+细辛（散风寒）+黄芩（清内热）+黄芪（固表） |
| 术附汤证 | 脾肾阳虚+风邪入脏 | 头重眩+苦极+不知食味 | 白术+附子+炙甘草+姜枣（温太阴少阴+益精气） |

### 8.11 历节病（类痛风）

| 维度 | 内容 |
|---|---|
| 本质 | **肝肾阴虚+风邪入侵→关节湿热带或寒湿痹阻** |
| 湿热型 | 关节剧痛肿大+黄汗+发热→桂枝芍药知母汤（桂枝麻黄防风解表驱风寒+白术祛湿+芍药知母滋阴清热+附子温通痹阻） |
| 寒湿型 | 节痛不可屈伸+寒象明显+脉沉紧→乌头汤（麻黄散寒+乌头逐少阴风寒+芍药甘草缓解拘急+黄芪益气扶正+白蜜制乌头毒性） |
| 鉴别 | 历节病（发热+疼痛+黄汗）vs 黄汗病（不发热+全身黄汗+脚冷+不痛） |
| 病因 | ①汗出入水中 ②饮酒湿热 ③过食酸咸伤肝肾 + 汗出当风 |
| 乌头用法 | 乌头（附子之母）辛热走窜，入少阴肾经搜逐风寒。⚠️ 必须白蜜二升煎乌头——蜜以甘润制其暴烈、使药性流连筋骨。"不知，尽服之"——未好转则继续喝。 |

引用来源：`2022-11-27_金匮5-3历节病的治法.pdf`

### 8.12 血痹（四肢麻痹）

| 维度 | 内容 |
|---|---|
| 本质 | **阳气阴血不足→风寒入侵四肢末端**——非大血管阻塞，非脑病 |
| 脉象 | 寸口关上微（阳气虚+阴虚血）+尺中小紧（寒邪入里入下焦）——微涩小紧 |
| 外证 | 身体不仁（麻木不知痛痒）——非历节关节疼痛 |
| 核心方 | 黄芪桂枝五物汤（桂枝汤去甘草+黄芪） |
| 去甘草理由 | 甘草甘缓缓和药力——此处需要驱邪，不能缓和。麻黄汤/桂枝汤/甘草泻心汤用甘草是让药力缓和留驻。 |
| 黄芪精确场景 | 肌肤层阳血气弱时用（邪已去大半）。若邪未去（表邪强盛），先用麻黄桂枝驱邪→邪去后再用黄芪巩固。 |
| ⚠️ 用药提醒 | 桃仁/红花/地龙等寒凉活血药不适合血痹——血痹本质是阳气血虚+风寒入末梢，而非热瘀。当归四逆汤温通血脉更合思路。 |

引用来源：`2022-12-04_金匮6-1血痹的治法（类四肢麻痹）.pdf`

### 8.13 虚劳（六经失调的累积终点）

| 维度 | 内容 |
|---|---|
| 本质 | 阴血亏损+阳气散失、浮散于外 |
| 脉诊总纲 | 脉大（假象·按下中空=阴虚阳亢）→脉极虚（按无力）→脉沉小迟（中气/脾阳脱）→脉革（弦大中空=外寒+阴血虚） |
| 阴阳两虚型 | 桂枝加龙骨牡蛎汤（心神不宁/失精/梦交）——有表证者在桂枝汤基础上加龙牡镇心 |
|  | 小建中汤（里急/悸/衄/腹痛/四肢酸痛/手足烦热/咽干口燥）——芍药加倍补阴血+温中缓急 |
|  | 天雄散（少阴寒证极重+无厥阴血热）——天雄逐寒+白术桂枝温中+龙骨镇心 |
| 阴虚为主型 | 八味肾气丸（阴虚血热+水湿瘀热+少量附子桂枝反佐）——⚠️ 不以"补肾阳"理解，反佐才是核心法度 |
|  | 薯蓣丸（肝血亏损+血虚血热+外有风寒）——厥阴+太阴+太阳多经合治的样板方 |
|  | 酸枣仁汤（仅心血虚热失眠）——⚠️ 90%现代人失眠是阳虚水湿，非此证 |
|  | 大黄蟅虫丸（内有干血+肌肤甲错+两目黯黑）——虫药搜邪+凉血活血，丸剂缓治 |
|  | 炙甘草汤（大补阴血+补心阳+脉结代）——生地黄16两+桂枝3两反佐精妙 |
| ⚠️ 核心提醒 | 90%现代人（饮食寒凉+生活习惯）是**阳虚水湿+肝郁+轻度血虚血热**，非单纯阴虚血热。所有虚劳方须在六经辨证下选用——先顾太阴少阴（温中祛湿）+再治厥阴（疏肝凉血），不从者必败。 |

### 8.14 新冠（时疫）的六经路径

| 维度 | 内容 |
|---|---|
| 邪气本质 | **太阴寒湿底+太阳膀胱经被封闭+热不得出**（非"风热"） |

| 痒的机理 | 风强+瘀热+湿热→隐疹→痒（阳虚邪重+血虚血燥）→痒抓破结痂癞 |
| ⚠️ 重要 | 表实有汗(表虚)→防己黄芪；表实无汗有热→越婢；表实无汗无热→甘草麻黄汤 |

### 8.28 皮水（金匮14-2 体内水饮溢出·无风邪）

### 8.29 正水与石水（金匮14-3 水在胸腹vs水在少腹）

| 维度 | 正水 (水在胸腹) | 石水 (水在少腹) |
|---|---|---|
| 本质 | 太阴脾经寒湿+水停胸腹 | 肾阳亏损+水气结于少腹 |
| 脉象 | 沉迟 | 自沉 |
| 核心症状 | 腹满+**喘**（水气上逆犯肺）→咳而喘+不渴 | 腹满+**不喘**（水在少腹不入胸）+脐肿+腰痛不得溺+足逆冷+面反瘦 |
| 主要脏器 | 脾+肺+胸腹腔体 | 肾+肝+少腹腔体 |
| 治疗总则 | 腰以下肿→利小便；腰以上肿→发汗 | 温肾利水 |
| 核心方 | 枳术汤（心下坚如盘+水饮——枳实行气破气泄水+白术健脾） | 肾气丸/四逆类 |
| 从表治 | 咳而喘+不渴+**有水肿**→按皮水发汗 | — |
| ⚠️ 禁忌 | 渴而下利+小便数者→**不可发汗** | — |
| 重要鉴别 | ③个痰饮方vs枳术汤：小半夏加茯苓(呕+心下痞+眩悸) / 甘遂半夏汤(脉伏+利后快仍满) / 小青龙(中寒+风寒+饮) / **枳术汤**(气滞水湿+腹皮硬+无痰饮) | 肝水(腹大不能转侧+胁下腹痛+小便续通) vs 肾水(腹大脐肿+不得溺+足逆冷) |

⚠️ **核心病理启示：** 很多水肿、痰湿的假象本质上是厥阴病症的上逆和血热——怕风冷手脚厥冷不是主证，肝经郁结+血热上逆才是。很多中老年病症关键在这里。

### 8.30 黄汗（金匮14-4 汗出如黄柏汁·阴血瘀热）

### 8.31 水气病血分与气分辨证（金匮14-5 深层辨证）

| 维度 | 内容 |
|---|---|
| 血分本质 | **血不利则为水**——阴血（经水/精液）不能正常回流入肾→肾经津液蒸腾障碍→水湿泛滥。**先经水断→后病水=血分（难治）** |
| 气分本质 | **寒水相搏+营卫俱劳**——阳气不足+寒邪水邪互结→手足逆冷+腹满肠鸣+身冷骨痛。**先病水→后经水断=水分（易治）** |
| 脉象总则 | 寸口沉而迟（沉=水·迟=寒）+趺阳伏（水谷不化）+少阴脉细（血少）+少阳脉卑（三焦决渎失常）→血分渐成 |
| 气分方 | **桂枝去芍药加麻辛附子汤**（桂枝+甘草+姜枣+麻黄+细辛+附子）：麻桂强力解表寒+生姜草枣温脾+细辛附子大温肾阳化水湿。当汗出如虫行皮中即愈 |
| 辨证案例 | 水气病多年→面部身体四肢皆肿+小便不利→主诉反言"胸中痛+气上冲咽+状如炙肉+咳喘"→误用十枣汤(下法)+误吐→胃家虚烦+咽燥欲饮+水谷不化+浮肿加重→⚠️这是医源性伤害 |
| 虚寒血热难治 | 血分治法两难：**温阳则加重血热→凉血活血则伤阳**——阴虚血热又阳虚水湿的困境 |

### 8.34 酒疸·黄疸诸方（金匮15 黄疸·阳明湿热系列）

|---|---|---|
| 栀子大黄汤 | 酒疸+心中懊憹或热痛 | 栀子(湿热瘀血)+大黄(泻下)+枳实(破气结)+豆豉(宣上) |

### 7.32 六经严格对应十二经络（核心框架确认）

黎明素问2023-11-18文章最终确认：**六经辨证中的六经严格对应十二经络**，这是理解所有方剂机制的根本。

| 经 | 足经 | 手经 | 升降 | 经脉功能 |
|---|---|---|---|---|
| **太阴** | 足太阴脾经 | 手太阴肺经 | ⬆升 | 腹部寒湿+水饮+咳嗽 |
| **少阴** | 足少阴肾经 | 手少阴心经 | ⬆升 | 肾阳温煦+心神 |
| **厥阴** | 足厥阴肝经 | 手厥阴心包经 | ⬆升 | 阴血调配+郁结 |
| **阳明** | 足阳明胃经 | 手阳明大肠经 | ⬇降 | 消化+大便 |
| **太阳** | 足太阳膀胱经 | 手太阳小肠经 | ⬇降 | 散热+水代谢 |
| **少阳** | 足少阳胆经 | 手少阳三焦经 | ⬇降 | 相火下降+气机通调 |

**三组循环三上三下：** 脾(升)-胃(降)一组 / 肾(升)-膀胱(降)一组 / 肝(升)-胆(降)一组。三阴升+三阳降=完整循环。

**经络比喻：** 经=高速公路·络=田间小道·**脏腑=高速服务区**。以脏腑论看病一定看不准。

**六经辨证流程（重申）：** 罗列六经全部症(30+项)→归类到对应经脉→找到1-2根重点→对应用药→忘记病名。

**⚠️ 太阴脾经的判断标准：** 现在全世界99%亚健康人都有太阴脾经寒湿。同时有舌苔水滑/齿印/大便粘马桶/手脚冷/睡眠不深/尿频/眼袋/小腿水肿中的3项以上就可下判断。

引用来源：`2023-11-18_六经辨证框架_绝离不开十二经络.pdf`

### 7.33 "补"是当今中医六经辨证的大忌（2023-11-24核心确认）

黎明素问2023-11-24文章总结：**六经辨证的核心策略只有2个——清除垃圾、疏通经脉。** 绝大部分人完全不需要"补药"。

| 经 | 实际方案 | 不需要补药的理由 |
|---|---|---|
| **太阴脾经** | 温中(姜枣甘)+祛湿(白术茯苓)+少数党参(需白术茯苓配合) | 党参/生晒参会加重湿气，多数情况下不常用！
| **厥阴肝经** | 疏肝(柴胡)+凉血(白芍/熟地黄/六味方根)+活血化瘀(重症) | 不是"补血"而是凉血、祛瘀活血。阿胶龟板鳖甲多数用不上
| **阳明胃经** | 降逆(半夏)+疏肝(柴胡)+清热(白虎/桂枝泽泻) | 不可能存在补药。枳实厚朴通便
| **太阴肺经** | 人参败毒散/杏苏散方根(柴胡枳壳桔梗前胡)+咳血苇茎汤 | 不能用药性滋腻的山药(说补肺不准确)
| **少阴肾经** | 制附片为唯一用药+干姜炙甘草 | **肾阳不能"补入"**——外界药物补不进去！附子=破冰寒+助阳气升腾
| **太阳膀胱经** | 杏仁紫苏叶葛根麻黄桂枝解表 | 没有补药
| **少阳胆经** | 已很少见(因饮食寒凉) | —

**"补药"的真相：** 阿胶/龟胶/枫斗/熟地=寒凉→越吃越严重。三七=温热但血虚热者不能用。安宫牛黄/片仔癀/铁皮枫斗/连花清瘟=绝对寒凉。**⚠️舌苔水滑+大便易溏者绝对不能用。** 党参/生晒参/红枣=带滋腻特性不能多用，否则加重湿气。

引用来源：`2023-11-24_补_是当今中医开方大忌.pdf`

### 7.35 真武汤的方根拆解示范（六经辨证×方根法）

传统教材对真武汤的描述：‘阳虚水泛：畏寒肢厥+小便不利+心下悸动+头目眩晕+身瞤动+沉重疼痛+浮肿+腰以下甚+腹痛泄泻+咳喘呕逆+舌胖齿痕苔白滑+脉沉细’——背诵难度极高。

**方根拆解后一目了然：**

| 方根 | 药物 | 作用经 | 针对症状 |
|---|---|---|---|
| ①② | 白术、茯苓 | 太阴脾经水湿 | 舌苔水滑+齿印+大便易溏+水肿 |
| ③ | 生姜 | 太阴脾经寒凉 | 胃寒+不渴+舌白 |
| ④ | 附子 | 少阴肾经虚寒 | 四肢厥逆+精神萎靡+恶寒下利 |
| ⑤ | 白芍 | 厥阴肝经血虚 | 身瞤动+站立不稳+头晕 |

**方根拆解后的灵活变通：**
| 变换 | 说明 |
|---|---|
| 生姜→干姜 | 有汗出怕热(厥阴阳明虚热)时换。干姜守而不走vs生姜走而不守 |
| 白术2两→4两 | 水湿重可加量，炙甘草少放（滞留水湿） |
| 附子3两→2-4两 | 根据少阴虚寒程度调整 |
| +桂枝（等量白芍） | 有鼻塞有汗=太阳表证+桂枝汤方根 |
| +紫苏叶/羌活独活/少量麻黄 | 有鼻塞无汗=太阳表寒 |

### 7.36 六经讲解顺序（与伤寒论原文不同的重要调整）

黎明素问提出：当前时代与张仲景时代不同，六经辨证的讲解顺序应调整：

| 顺序 | 经 | 理由 |
|---|---|---|
| **1** | **太阴脾经** | **核心！** 饮食寒凉导致绝大多数人太阴脾经寒湿→其他所有病症根源 |
| **2** | 阳明胃经+厥阴肝经 | 太阴不升→阳明不降；太阴寒湿→肝经郁结→厥阴病 |
| **3** | 少阴肾经+太阳膀胱经 | 脾阳虚→肾阳亦虚；肾阳虚→膀胱经气不足→易外感 |
| **4** | **少阳胆经** | 现代人寒湿多+厥阴病多→少阳热症已很少见 |
| 附带 | 手太阴肺经方根方药 | 肺经咳嗽痰饮常与脾经同调 |

⚠️ **注意：** 《伤寒论》邪气从外向内（太阳→阳明→少阳→太阴→少阴→厥阴），但**临床中太阴寒湿是最核心的证型**，因此辨证和教学应先从太阴入手。

引用来源：`2023-11-30_六经_方根的重要意义.pdf`、`2023-11-26_中医入门框架_六经辨证_总目录.pdf`

### 7.37 鼻炎三经同治（太阳+太阴+肺经）

黎明素问对鼻炎的理解：**鼻炎不是鼻子的问题，是三条经脉协同失调。**

| 经 | 角色 | 辨证要点 | 方根 |
|---|---|---|---|
| **足太阳膀胱经**（主表） | 表层风寒入侵 | 受凉加重·早起鼻塞（睡觉时阳气入里·表卫减弱） | 桂枝汤（有汗）/麻黄（无汗）/紫苏叶/羌独活 |
| **足太阴脾经**（主湿） | 脾经寒湿化生的痰涎上泛 | 舌苔水滑齿印+大便粘+白痰多 | 白术茯苓干姜炙甘草 |
| **手太阴肺经**（窍在鼻） | 肺经风寒导致气血供给不畅 | 白鼻涕泡沫多·鼻塞·打喷嚏 | 细辛干姜（强力祛白痰）+解表药 |

**治疗周期：** 2-3天显著好转。⚠️根本在脾经寒湿（受饮食影响），恢复较慢。脾经寒湿清除差不多则鼻炎可彻底治愈。

**睡觉时鼻塞加重机制：** 人睡眠→太阳经阳气回收入里→表卫减弱→风寒更易入侵→鼻涕鼻塞加重。同理：血热之人晚上睡觉时阳气入里→燥热盗汗；阳明瘀热小孩→蹬被易感冒。

引用来源：`2023-12-10_微信_教学群_学员问答20231210.pdf`

### 7.38 桂枝汤方根辨析（发汗 vs 敛汗）

传统教材对桂枝汤的阐释模糊：「解肌发表、调和营卫」——初学者无法理解。

| 方根 | 作用方向 | 各司其职 |
|---|---|---|
| 生姜+桂枝（不含白芍） | **发汗**（加强体表驱寒） | 身体外层皮肤的兵力能量充沛了，驱寒驱邪的力量也加强了 |
| 白芍 | **收敛**肝经过度排汗液·**凉血**回收血液 | 收敛虚汗——使身体不过度出汗 |
| 姜+枣+草 | 温太阴脾经·**提供体液** | 桂枝发汗过度消耗的能量由姜枣草补充 |

**桂枝汤核心机制：** 桂枝+白芍同时使用，身体自判是否需要出汗。
- 需出汗→再出一次，排出寒邪（此时喝热粥覆被）
- 需收汗→白芍收敛，汗自止

**身体自调而非药物主宰：** 与失眠「安神」同理——恢复六经顺畅→该睡则睡，该吃则吃，该出汗则出汗。

引用来源：`2023-12-10_微信_教学群_学员问答20231210.pdf`

### 7.39 "安神"的六经辨证本质

**根本没有安神药这种东西。** 失眠是六经不畅导致的症状表现，不是独立的病。

| 失眠·安睡 | 六经辨证解释 |
|---|---|
| 为什么会失眠 | 六经气机不畅→该升不升·该降不降→睡眠不能正常发生 |
| 为什么能睡着 | 六经恢复顺畅→身体自功能恢复→该睡就会睡 |
| 案例 | 某学员方根（术苓姜草+柴芍+厚朴+熟地山萸）无一味安神药→当晚见效 |

**⚠️ 临床教训：** 凡是给患者开「安神药」（酸枣仁、柏子仁、远志、龙骨牡蛎）的医生，说明不懂六经辨证。恢复六经正常运行后，失眠自愈。

引用来源：`2023-12-10_微信_教学群_学员问答20231210.pdf`

### 7.40 牙齿问题=厥阴肝经为主（2023-12临床确认）

黎明素问在12月教学群确认：**牙齿问题多数与厥阴肝经有关**（出血/血虚层面），而非传统认为的少阴肾经。

| 经 | 牙痛机制 | 治疗方根 |
|---|---|---|
| **厥阴肝经**（主要） | 牙龈出血·牙浮肿·牙酸痛·唇干脱皮·口腔溃疡——血虚层面 | 柴胡（3）+白芍（3-6-9）→1-2剂见效 |
| 少阴肾经 | 阳虚下元虚寒+牙齿松动剧烈——少见且不与厥阴肝经牙痛同时存在 | 附子→需六经综合辨证 |

**鉴别口诀：** 少阴牙痛阳虚（精神萎靡·四肢厥逆·下利清谷），厥阴牙痛血虚（牙龈出血·夜醒·皮肤痒·月经紊乱）。

引用来源：`2023-12-13_微信_教学群_学员问答20231213.pdf`

### 7.41 耳鸣六经辨证案例（水湿肺热→葶苈大枣泻肺汤）

黎明素问在2023-12-14教学群展示的完整辨证过程：

**案例主诉：** 耳鸣，睡觉时/醒来时加重，白天减轻，伴随眼白红血丝+躺下时爆咳。

**辨证思路：**
| 征象 | 六经分析 |
|---|---|
| 耳鸣白天轻·睡前/醒后加重·躺下爆咳+眼白红丝 | → **肺中有水湿**（水湿随体位流动→睡下加重）·水湿瘀久→肺热（眼白红丝） |
| 最直接推论：水湿肺热 | → 水湿在肺瘀堵，影响阳气流通→耳鸣 |
| 舌苔水滑·齿印·便粘·手脚冷 | → 太阴脾经寒湿（基础证） |
| 精神萎靡/怕冷 | → 少阴肾阳虚（兼证） |
| 爱生气/郁闷/月经不调 | → 厥阴肝经郁结+轻血虚 |

**治疗方药（7-8味）：**
| 经 | 方根 | 药物 |
|---|---|---|
| 少阴 | 附子（少量） | 制附片3-6-9 |
| 太阴 | 术+苓+姜+草 | 白术6+茯苓9+干姜6+炙甘草6 |
| 水湿肺热 | 葶苈大枣（核心方根） | 葶苈子9+大枣5枚（《金匮》方：祛肺中水湿瘀热） |
| 厥阴 | 疏肝+凉血 | 柴胡3+白芍6 |

**治疗过程：**
1. 首方用上→2天略有好转
2. 去葶苈子+加紫苏叶（解残余表寒）→去白芍
3. 再去附子+加荆芥薄荷（清阳明余热）

**⚠️ 核心启示：** 耳鸣不是独立病名，不可对应「肾虚」「肝火」。必须按六经辨证罗列全部症状，找到对应经→选择相应方根。

引用来源：`2023-12-14_微信_教学群_学员问答20231214.pdf`、`2023-12-11_常见方根的用量.pdf`
