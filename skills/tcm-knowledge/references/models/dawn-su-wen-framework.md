## 一、根本命题

**《伤寒论》398 条不是方剂分类手册，而是邪气从体表到内里的「进军路线图」。** 每一经对应一条正经，每一证型对应这条经上的一个受阻位置。

传统模型把六经当作并列的"六大证候群类型"来分类；黎明素问模型把六经理解为**六条通路的水位观察口**——邪气走了哪条路、堵在哪一段、水的升降流动出了什么问题。

引用来源：`2022-10-03_《伤寒论》的中医解剖原理（绪论）.pdf`

## 二、核心框架：一大循环 + 两条轴

### 人体大循环 = 水升火降（阴升阳降）

```
         ⬆ 少阳相火（热量）从顶向下敛降
         │
  太阳 ← ← 阳明 ← 少阳    ← 三阳（阳降：热量回收）
    ↓                        ↑
  体表散热 ← ← ← ← ← ← ← 热量向上扩散
    ↓                        ↑
  少阴 → 太阴 → 厥阴    → 三阴（阴升：水气升发）
         │
         ⬇ 肝木（水气）从底向上生发
```

### 关键差异：三阳不是"升"，三阳都在"降"

| 观点 | 传统教材常见表述 | 黎明素问 |
|---|---|---|
| 少阳 | 春升之气，疏泄条达 | 相火（热量）从顶向下敛降 |
| 太阳 | 人体最大阳经 | 体表散热通道——热量从内向外散发 |
| 阳明 | 多气多血 | 热量从外向下向内收敛 |
| 三阴 | 里虚寒证 | 水气（阴）从下向上蒸腾 |

**水升火降两个方向：**
- **水升（阴升）**: 少阴(水源) → 太阴(水运) → 厥阴(输布全身)
- **火降（阳降）**: 少阳(热量开始下降) → 太阳(体表发散) → 阳明(向内向下收敛归零)

> 引用来源：`2022-10-03_《伤寒论》的中医解剖原理（绪论）.pdf`、`2023-11-03_中医入门框架《六经辨证》总目录.pdf`

## 三、六经的经络解剖定位

### 三阳经——循行路线与病理对应

**1. 太阳病——足太阳膀胱经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于目内眦→上行巅顶→下项→挟脊→抵腰→走背→下至足 |
| 病理对应 | 头项强痛（后脑/头顶痛）、项背强几几、腰背痛、发热恶寒 |
| 核心病机 | 经气被寒邪束住——膀胱经不走气了→体表散热受阻→郁而发热 |
| 代表方剂 | 麻黄汤（寒闭——经气完全不通）、桂枝汤（风袭——经气部分不通） |

**2. 阳明病——足阳明胃经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于鼻旁→环唇→下颊→循喉咙→下膈→属胃→下行至足 |
| 病理对应 | 全身蒸蒸汗出、前额痛、口干舌燥、大便硬结 |
| 核心病机 | 热量向下向内收敛的通道被堵→胃中瘀热 |
| 代表方剂 | 白虎汤（气分热）、承气汤类（腑实热） |

**3. 少阳病——足少阳胆经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于目外眦→绕耳前后→下胸中→循胁→下行至足 |
| 病理对应 | 两侧头痛、口苦咽干目眩、胸胁苦满、寒热往来 |
| 核心病机 | 相火从顶向下敛降遇到障碍→热卡在胆经两侧 |
| 代表方剂 | 小柴胡汤（疏通胆经瘀热——柴胡+黄芩清上端，半夏+生姜降下端） |

### 三阴经——递进演变关系（与传统最大不同）

**黎明素问认为三阴病是递进演化的，不是并列的。**

```
太阴寒湿 ──恶化──→ 少阴寒+上热 ──恶化──→ 厥阴寒热交替
(吐利、腹满)      (脉微细、但欲寐、咽痛)     (消渴、气上撞心、厥热胜复)
```

**4. 太阴病——足太阴脾经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于足大趾→上行下肢内侧→入腹→属脾络胃→上行至舌 |
| 病理对应 | 腹满而吐、食不下、自利益甚、时腹自痛、舌苔白腻水滑 |
| 核心病机 | 脾阳虚衰→寒湿内生→水气升发无力 |
| 临床地位 | **太阴病是一切六经病的前提和基础。** 作者认为99%现代人都有不同程度的太阴虚寒——冷饮、牛奶、水果、空调是根本原因。 |
| 代表方剂 | 理中汤（太阴虚寒吐利）、四逆汤（恶化为少阴时） |

**5. 少阴病——足少阴肾经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于足小趾下→斜走足心→上行下肢内侧后缘→属肾络膀胱→循喉咙→挟舌本 |
| 病理对应 | 脉微细、但欲寐（精神萎靡）、咽痛（肾经走喉咙）、下利清谷、四肢厥逆 |
| 核心病机 | 太阴病恶化→肾阳亦衰→水寒于下+虚热浮于上（上热下寒） |
| 与其他经关系 | 少阴是**三阴病的中间阶段**——上承太阴（寒湿输布至此）、下启厥阴（寒化极则入厥阴） |
| 代表方剂 | 四逆汤（少阴虚寒）、白通汤（虚阳浮越）、黄连阿胶汤（阴虚火旺咽痛） |

**6. 厥阴病——足厥阴肝经**

| 项目 | 内容 |
|---|---|
| 循行 | 起于足大趾上→上行下肢内侧→绕阴器→抵小腹→属肝络胆→上行至巅顶 |
| 病理对应 | 消渴、气上撞心（肝气冲心/胃——当归四逆汤证）、心中疼热、饥不欲食、厥热胜复 |
| 核心病机 | 最深层阴寒→阳气与寒邪在肝经交战→交替出现寒与热（寒热错杂） |
| 关键鉴别 | 厥阴病与少阴病的咽痛鉴别——少阴咽痛是肾经虚火上浮（尿清长、下肢冷、脉沉微）；厥阴咽痛是肝经瘀热上冲（小便黄、大便干、咽干口苦、脉弦数） |
| 代表方剂 | 乌梅丸（厥阴寒热错杂）、当归四逆汤（血虚寒凝+肝经顶端瘀热） |

> 引用来源：六篇核心 PDF（2022-10-03 六经 @ 太阳/阳明/少阳/太阴/少阴/厥阴）

## 四、中药的"经络路径"重新理解

黎明素问对中药的理解不按"功效分类"走，而是按**"这味药作用于哪条经的哪个位置、帮助了哪个方向的循环"**来理解。

### 核心原则

每种药在体内有明确的**经络路径轨迹**——沿某条经的某个部位起作用，促进或抑制"水升火降"的某个环节。

### 代表药物重新理解

#### 柴胡
| 维度 | 内容 |
|---|---|
| 教材定义 | 疏散退热、疏肝解郁、升举阳气 |
| 黎明素问 | **通胆经上端瘀热、助胆火下降**——胆经从上向下走，第一道关口就是柴胡证（胸胁满、口苦）。柴胡不是"升"而是"通"——把胆经上端的热"顺"下去。 |
| 炮制观点 | 支持生用（醋酸制后效力大减） |
| 引用 | `2024-05-18_【柴胡】疏肝解郁.pdf` |

#### 白芍
| 维度 | 内容 |
|---|---|
| 教材定义 | 养血柔肝、平肝止痛、敛阴止汗 |
| 黎明素问 | **凉血柔胆、助胆火下降**——黄芩清胆经气分热（上端）、白芍清胆经血分热（助热从胆经下端向下走）。白芍与柴胡是胆经药对——柴胡通上端、白芍凉下端。 |
| 引用 | `2024-05-27_【白芍】平肝降胆.pdf` |

#### 乌梅
| 维度 | 内容 |
|---|---|
| 教材定义 | 敛肺、涩肠、生津、安蛔 |
| 黎明素问 | **清肝经上端瘀热**——厥阴肝经顶端瘀热上冲致消渴、气上撞心。乌梅在乌梅丸中负责清理这个位置的瘀热，再给干姜附子往下托底。 |
| 引用 | `2024-05-28_当归四逆汤中的乌梅作用推测.pdf`（未见，推测自上下文） |

#### 葛根
| 维度 | 内容 |
|---|---|
| 教材定义 | 解肌退热、透疹、生津止渴、升阳止泻 |
| 黎明素问 | **透阳明经初热，深入肌肉下层搜刮寒邪瘀热**——葛根治的是阳明经初起的寒邪瘀热（葛根汤证/桂枝加葛根汤证）。作用点在肌肉层（足阳明经循行处）。 |
| 引用 | `2024-06-04_【葛根】透阳明初热.pdf` |

#### 石膏
| 维度 | 内容 |
|---|---|
| 教材定义 | 清热泻火、除烦止渴 |
| 黎明素问 | **泻胃火透肺热，不可持续使用**——生石膏治的是阳明经的气分大热（白虎汤证），热退即停，不可久用。石膏的"透"体现在阳明胃经的郁热从肌肉毛孔向外透发。 |
| 引用 | `2024-04-24_【阳明】生石膏，切不可持续使用.pdf` `2024-07-19_【生石膏】泻胃火、透肺热.pdf` |

#### 附子
| 维度 | 内容 |
|---|---|
| 教材定义 | 回阳救逆、补火助阳、散寒止痛 |
| 黎明素问 | **少阴经的核心药——从肾经下端烧起，热力沿经上升**。附子配干姜（四逆汤）——干姜守太阴中焦，附子走少阴下焦，热力从下向上沿肾经注入全身。⚠️ 附子毒性在剂量而非配伍——药典剂量标准过低，临床有效剂量需因人调整。 |
| 引用 | `2024-12-28_【附子】的使用和禁忌-少阴肾经的常用方根方药（2）.pdf` |

#### 半夏
| 维度 | 内容 |
|---|---|
| 教材定义 | 燥湿化痰、降逆止呕、消痞散结 |
| 黎明素问 | **降胃逆、祛湿痰**——作用于足阳明胃经（上端咽喉→中间胃→下端肠）。**降逆**是半夏在胃经上的核心功能。与黄连搭配（半夏泻心汤）治胃热夹湿，与干姜搭配治胃寒夹湿。 |
| 引用 | `2024-05-06_【半夏】降胃逆、祛湿痰.pdf` |

#### 黄芪
| 维度 | 内容 |
|---|---|
| 教材定义 | 补气升阳、固表止汗、利水消肿 |
| 黎明素问 | **补气不可滥用**——黄芪补的是太阴脾经的气（中焦之气），不是"全身的气"。脾虚（太阴寒湿）者用之效佳，但若患者上焦有热（肺热、心火）或下焦有湿热，黄芪会"壅补"——把热锁住不让出去。黎明素问批评现代中医过度使用黄芪，认为很多问题恰恰是滥用黄芪导致的。 |
| 引用 | `2024-04-30_【黄芪】黄芪补气，不可滥用.pdf` |

## 五、剂量观与安全哲学

| 维度 | 传统药典标准 | 黎明素问 |
|---|---|---|
| 剂量基准 | 药典固定区间（如附子3-15g） | 以患者为准——视邪气严重程度和体质灵活调整 |
| 东汉1两=？ | 主流有15g/13.8g/3g之争 | 反对15g原教旨主义——认为方子应根据患者体质和邪气强弱来调整，不是机械按比例换算 |
| 毒性管理 | 药典标准剂量 | 剂量的安全线取决于**患者的排邪通道是否通畅**——不是剂量本身，而是堵不堵 |
| 煎煮法 | 按药典炮制规范 | 支持生用部分药物（如生柴胡、生白芍）——认为炮制过度会损失药性 |

## 六、方根理论（与灵枢台拆方体系的映射关系）

黎明素问体系中的 **「方根」**（核心药对/药组）与灵枢台现有的拆方体系高度兼容，只是解释角度不同：

| 方根 | 教材解释 | 黎明素问解释 | 引用 |
|---|---|---|---|
| 柴胡+黄芩 | 和解少阳 | 柴胡通胆经上端、黄芩清胆经气分热 | 少阳经的疏通药对 |
| 芍药+甘草 | 缓急止痛 | 芍药凉血柔胆、甘草缓中——缓解胆经/肝经的痉挛 | 阴经经脉松弛药对 |
| 麻黄+桂枝 | 发汗解表 | 麻黄开太阳经气闭、桂枝通太阳经血行——两药配合把太阳经"打通" | `2023-11-28_看方子的窍门是方根.pdf` |
| 附子+干姜 | 回阳救逆 | 干姜守太阴中焦、附子走少阴下焦——热力从下向上沿肾经注入 | 少阴经回阳药对 |
| 半夏+黄连 | 辛开苦降 | 半夏降胃逆、黄连泻心火——在阳明经上形成一个"升降对" | 阳明经辛开苦降药对 |

## 七、与传统模型的主要差异对照表

| 维度 | 传统教材/药典模型 | 黎明素问模型 | 兼容策略 |
|---|---|---|---|
| 六经本质 | 六大症候群分类 | 六条正经的邪气路径 | 证型+路径双标注 |
| 三阴关系 | 并列分型 | 太阴→少阴→厥阴递进 | 在证候定义中加递进关系字段 |
| 太阳病定位 | 表证/风寒 | 体表散热通道不畅 | 核心症状一致，解释不同 |
| 少阳病 | 半表半里 | 胆经相火下降受阻 | 核心症状一致，解释不同 |
| 柴胡功效 | 疏肝解郁，升阳 | 通胆经、助胆火下降 | 在药物注解中并存两种解释 |
| 芍药功效 | 养血柔肝 | 凉血柔胆 | 同上 |
| 太阴病临床权重 | 非核心（六经之一） | **一切六经病的基础** | 在安全警告中加"太阴体质"提醒 |
| 剂量哲学 | 药典标准 | 因人灵活调节 | 剂量字段加"灵活调整"备注 |
| 方根学习 | 方剂学+药对 | 按经络路径理解药的作用位置 | 现有拆方体系可直接扩展路径注解 |
