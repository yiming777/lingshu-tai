# 白话主诉 → 四诊映射

> 灵枢台 · tcm-bianzheng 参考文件
> 当用户使用日常语言（非中医术语）时，加载本文件将主诉翻译为辨证方向
> 加载策略：Phase 1A 检测到白话输入 → `read` 本文件对应条目 → 生成四诊追问

---

## 使用说明

本文件把"用户说的大白话"翻译成"中医需要追问的方向"。每一条包含：

- **用户可能说的** → 白话表达变体
- **中医映射** → 涉及哪些脏腑、哪些病性可能
- **关键区分问题** → 用大白话追问，帮用户自己判断方向
- **指向证型** → 可能的证型范围（仅供参考，不预设结论）

---

## 一、消化系统

### 拉肚子 / 腹泻 / 老跑厕所

**中医映射**：脾、大肠、肾（五更泻）；寒湿、湿热、脾虚、肾阳虚

**关键区分问题**：
1. 大便稀到什么程度？像水一样（水样便）？还是只是比较软不成形？
2. 肚子疼不疼？疼的时候想不想捂着/暖和一下？还是一阵阵绞着疼？
3. 有没有没消化完的食物？有没有黏液或血？
4. 吃了什么之后会加重？油腻的？生冷的？
5. 这种情况多久了？几天（急）还是几个月/几年（慢）？
6. 是不是天快亮的时候必须跑厕所（五更泻）？

**指向证型**：
- 水样便+怕冷+不臭 → 寒湿困脾
- 大便黏臭+肛门灼热 → 大肠湿热
- 长期大便不成形+乏力+食欲差 → 脾气虚/脾阳虚
- 天亮前必泻+腰酸怕冷 → 脾肾阳虚（五更泻）

---

### 胃不好 / 胃痛 / 胃胀

**中医映射**：胃、脾、肝；寒、热、虚、气滞、食积

**关键区分问题**：
1. 是胀（感觉有气堵着）还是痛？还是又胀又痛？
2. 疼的时候喜欢用手按着/捂着热水袋（喜按喜温）？还是按了更疼（拒按）？
3. 和情绪有关吗？生气、紧张的时候会不会加重？
4. 什么时候疼？饭前（空腹疼）还是饭后？
5. 有没有反酸、烧心、打嗝？
6. 胃口怎么样？想吃但不敢吃？还是完全没胃口？

**指向证型**：
- 胀痛+嗳气+情绪有关 → 肝气犯胃（气滞）
- 灼痛+反酸+口苦 → 肝胃郁热
- 隐痛+喜按+空腹疼 → 脾胃虚寒
- 胀满+嗳腐+大便酸臭 → 食积

---

### 便秘 / 大便干 / 拉不出来

**中医映射**：大肠、胃、脾、肝、肾；热、气滞、气虚、血虚、阳虚

**关键区分问题**：
1. 大便干硬像羊粪？还是排便费力但大便不干？
2. 多久排一次？三天以上？
3. 口干口渴吗？喜欢喝热水还是冷水？
4. 肚子胀不胀？有便意但排不出？
5. 平时累不累？有没有气不够用的感觉？

**指向证型**：
- 大便干硬+口干口臭+面红 → 胃肠实热
- 排便费力+腹胀+嗳气 → 气滞便秘
- 大便不干但排不出来+乏力+面色白 → 气虚便秘
- 大便干+口干+手脚心热 → 阴虚便秘
- 排便困难+怕冷+腹冷痛 → 阳虚便秘

---

## 二、睡眠与精神

### 睡不着 / 失眠 / 睡不好

**中医映射**：心、肝、肾；火、阴虚、血虚、痰

**关键区分问题**：
1. 是入睡困难（躺下半天睡不着）？还是睡着了容易醒？还是天没亮就醒再也睡不着？
2. 做梦多不多？梦的内容吓人吗？
3. 心里烦不烦？有没有想很多事停不下来？
4. 白天累不累？有没有心慌、容易受惊吓？
5. 有没有口干、手心脚心热、盗汗（睡着后出汗）？
6. 和最近的烦心事有关吗？

**指向证型**：
- 入睡困难+心烦+口干+舌尖红 → 心火亢盛
- 多梦易醒+心慌+面色萎黄 → 心血虚/心脾两虚
- 入睡困难+口干+手心热+盗汗 → 阴虚火旺（心肾不交）
- 睡眠浅+易惊醒+胆小+口苦 → 心胆气虚
- 睡眠不安+胸闷+痰多+苔腻 → 痰热扰心

---

### 累 / 没精神 / 浑身无力

**中医映射**：脾、肺、肾；气虚、湿困、阳虚、血虚

**关键区分问题**：
1. 累的感觉——是动一动就喘不上气？还是身体沉重像绑了沙袋？还是懒洋洋不想动？
2. 是整天都累，还是早上起来就累？下午加重？
3. 吃饭怎么样？胃口好吗？
4. 怕冷还是怕热？
5. 脸色怎么样？偏白还是偏黄？
6. 有没有浮肿？腿肿？眼皮肿？

**指向证型**：
- 乏力+气短+易感冒+声音低 → 肺气虚
- 乏力+食欲差+大便不成形 → 脾气虚
- 乏力+身体沉重+头昏蒙蒙+舌苔腻 → 湿困脾胃
- 乏力+怕冷+四肢不温 → 阳虚
- 乏力+面色苍白+心慌+头晕 → 血虚

---

## 三、寒热与疼痛

### 手脚冷 / 怕冷

**中医映射**：肾、脾；阳虚、气血不足、阳气郁闭

**关键区分问题**：
1. 是全身怕冷，还是光手脚冷？手脚冷的范围到哪里（手腕脚踝？还是过肘过膝？）
2. 冷的时候暖一暖能暖和过来吗？还是一直冷？
3. 是夏天也怕冷，还是只有冬天？
4. 有没有腰酸？夜尿多吗？
5. 心情怎么样？有没有情绪低落、胸闷？

**指向证型**：
- 全身怕冷+腰酸+夜尿多 → 肾阳虚
- 怕冷+肚子凉+大便稀 → 脾阳虚
- 手脚冷但身体不冷+情绪低落+脉弦 → 阳气郁闭（四逆散证）
- 手脚冷+面色白+心慌+头晕 → 血虚寒厥（当归四逆汤证）

---

### 上火 / 口疮 / 长痘 / 口干

**中医映射**：心、肝、胃、肺；实火、虚火

**关键区分问题**：
1. 哪里上火？嘴巴（口疮/溃疡）？脸上（痘痘）？喉咙？眼睛？
2. 口疮疼不疼？周围红不红？是单个大泡还是散在小泡？
3. 痘痘长在哪里？额头（胃）？脸颊（肺）？下巴（肾/内分泌）？
4. 口干想喝水吗？喜欢冷水还是热水？
5. 有没有手心脚心热？晚上出汗？
6. 大便情况？干还是稀？

**指向证型**：
- 口疮红肿疼痛+口臭+便秘 → 胃火（实火）
- 口疮反复发作+口干+手心热 → 阴虚火旺（虚火）
- 痘痘红肿+面油+口苦+大便黏 → 湿热
- 口干+眼干+咽干 → 肺胃阴虚/燥邪

---

### 头痛

**中医映射**：外感（风寒/风热）、肝、肾、气血；经络定位

**关键区分问题**：
1. 头疼在哪里？前额？两侧？头顶？后脑勺？整个头都疼？
2. 疼是什么感觉？胀痛？刺痛？一跳一跳的疼？像箍了紧箍咒？
3. 什么时候加重？累了？生气了？吹风了？月经前后？
4. 有没有恶心、眼花、怕光、怕吵？

**指向证型 + 经络定位**：
- 后脑勺痛+怕风+脖子僵 → 太阳经（外感风寒）
- 前额痛+眉棱骨痛 → 阳明经（胃热/外感）
- 两侧痛+偏头痛+口苦 → 少阳经（肝火/肝阳）
- 头顶痛+干呕+吐涎沫 → 厥阴经（肝寒）
- 整个头痛+胀痛+面红+急躁 → 肝阳上亢
- 头痛隐隐+劳累加重+面色白 → 气血虚

## 四、呼吸系统

### 感冒 / 发烧 / 咳嗽

**中医映射**：肺、卫分；风寒、风热、表里

**关键区分问题**：
1. 怕冷还是怕热？怕冷严重吗？发烧体温多少？
2. 出汗了吗？一点汗没有，还是微微出汗？
3. 咳嗽有没有痰？痰是白的稀的还是黄的稠的？
4. 嗓子痒还是嗓子疼？鼻子是流清鼻涕还是黄鼻涕？
5. 全身疼不疼？特别是后脑勺和后背？

**指向证型**：
- 怕冷严重+无汗+全身疼+脉浮紧 → 风寒表实（麻黄汤证）
- 怕风+有汗+发热不高+脉浮缓 → 风寒表虚（桂枝汤证）
- 发热重+咽痛+痰黄+口渴 → 风热犯肺（银翘散证）
- 干咳+咽干+痰少难咳+发热轻 → 燥邪伤肺（桑杏汤证）

---

### 老咳嗽 / 咳嗽老不好

**中医映射**：肺、脾、肾；痰湿、阴虚、气虚

**关键区分问题**：
1. 咳了多久了？几周（急）还是几个月（慢性）？
2. 干咳还是有痰？痰什么颜色？稀的还是稠的？容易咳出来吗？
3. 什么时候咳得厉害？早上？晚上躺下？还是活动后？
4. 有没有喘？气够不够用？
5. 抽烟吗？做什么工作？

**指向证型**：
- 干咳无痰+咽干+午后加重 → 肺阴虚
- 咳白稀痰+怕冷+遇冷加重 → 寒饮伏肺
- 咳黄稠痰+胸闷+发热 → 痰热壅肺
- 咳痰多白黏+乏力+食欲差 → 脾虚痰湿
- 久咳+气短+动则喘 → 肺肾气虚（肾不纳气）

---

## 五、妇科

### 月经不调 / 痛经

**中医映射**：肝、脾、肾、冲任；寒、热、气滞、血瘀、血虚

**关键区分问题**：
1. 月经周期——提前？推后？还是忽前忽后没规律？
2. 量多还是量少？颜色鲜红、暗红、还是有血块？
3. 疼在哪里？小腹正中？两侧？腰？
4. 疼的时候喜欢捂着/暖着（喜温喜按）？还是不能按（拒按）？
5. 来月经前乳房胀不胀？情绪好不好？
6. 有没有白带？什么颜色、气味、量？

**指向证型**：
- 经前/经期小腹胀痛+血块+块出痛减+情绪差 → 气滞血瘀
- 小腹冷痛+喜温喜按+经色暗+畏寒 → 寒凝血瘀
- 经后小腹隐痛+量少色淡+面色白 → 气血虚弱
- 经期提前+量多色红+口干 → 血热
- 月经忽前忽后+量少+腰酸+头晕 → 肾虚

## 六、其他常见主诉

### 腰疼 / 腰酸

**中医映射**：肾、膀胱经；虚、寒湿、血瘀

**关键区分问题**：
1. 是酸痛（隐隐约约）还是刺痛（固定位置）？
2. 是一直疼还是活动后加重？累了以后加重？
3. 怕冷吗？遇冷/阴雨天加重？
4. 有没有腿麻、腿疼？

**指向证型**：
- 腰酸隐隐+劳累加重+膝软+头晕 → 肾虚
- 腰冷痛+沉重+阴雨天加重 → 寒湿腰痛
- 腰痛固定+刺痛+有外伤史 → 血瘀腰痛

---

### 头晕 / 眩晕

**中医映射**：肝、脾、肾；风、火、痰、虚

**关键区分问题**：
1. 是感觉天旋地转（眩晕）？还是感觉头昏昏沉沉？
2. 发作的时候有没有恶心、耳鸣？
3. 是突然发作还是一直都有？和姿势变化有关吗（比如突然站起来）？
4. 血压高不高？

**指向证型**：
- 头晕+面红+急躁+口苦+血压高 → 肝阳上亢
- 头晕+胸闷恶心+痰多+身重 → 痰湿中阻
- 头晕+眼花+面色白+心悸 → 气血两虚
- 头晕+耳鸣+腰酸+健忘 → 肾精不足

---

### 皮肤痒 / 出疹子 / 湿疹

**中医映射**：肺（主皮毛）、肝、脾；风、湿、热、血虚

**关键区分问题**：
1. 疹子什么样？红的一片片？小水泡？干皮屑？
2. 痒的程度？越抓越痒？晚上痒得更厉害？
3. 什么部位？局部（哪里）还是全身？
4. 什么情况下加重？吃海鲜/辣的/喝酒？情绪不好？天气变化？
5. 大便情况？

**指向证型**：
- 疹红+痒+热感+遇热加重 → 风热
- 水泡+渗液+舌苔腻 → 湿热
- 皮肤干燥+脱屑+瘙痒+秋冬加重 → 血虚风燥
- 荨麻疹时隐时现+与情绪有关 → 风邪+肝郁

---

### 口臭 / 口苦 / 口黏

**中医映射**：胃、肝、胆；热、湿热、食积

**关键区分问题**：
1. 嘴里的感觉——苦？臭？甜？黏腻？没味道？
2. 早上起来特别明显吗？
3. 胃口怎么样？大便情况？
4. 有没有反酸、烧心？

**指向证型**：
- 口臭+牙龈肿痛+便秘 → 胃火
- 口苦+胁痛+急躁 → 肝胆火旺
- 口甜+口黏+脘痞+苔腻 → 脾胃湿热
- 口淡+食欲差+乏力 → 脾胃气虚

---

### 脱发

**中医映射**：肾（其华在发）、肝（藏血）、血

**关键区分问题**：
1. 脱发多久了？是突然大量掉还是一直稀疏？
2. 头发油不油？头皮痒不痒？
3. 有没有腰酸、怕冷、头晕？
4. 最近压力大吗？熬夜吗？

**指向证型**：
- 脱发+头发枯黄+头晕+面色白 → 血虚
- 脱发+腰酸+怕冷+耳鸣 → 肾阳虚/肾精不足
- 脱发+头皮油+痒+口苦 → 湿热
- 脱发+情绪差+失眠 → 肝郁
