# 运气维度调用规则

> 灵枢台 · tcm-bianzheng 参考文件
> 五运六气在辨证中的使用原则

---

## 什么情况下调用

| 场景 | 规则 |
|------|------|
| **时令病/外感病** | 必调用。外感与时气直接相关。 |
| **慢性病突然加重** | 调用，检查是否与运气转换节点（如客气转换）有关。 |
| **疫病相关辨证** | 必调用。《素问·刺法论》："天地迭移，三年化疫。" |
| **症状特征与当前运气格局典型病机高度吻合** | 调用进行相关性分析。 |
| **用户主动要求** | 调用独立五运六气分析（转 `/tcm-wuyun-liuqi`）。 |

---

## 什么情况下不调用

| 场景 | 规则 |
|------|------|
| **纯内伤病**（七情/饮食/劳倦为主要病因） | 不强行纳入运气框架。 |
| **急诊/创伤/中毒** | 不调用，优先紧急处理。 |
| **运气特征与当前证候明显不符** | 不强行解释，标注"与当前运气格局无显著关联"。 |

---

## 推算方法（内嵌）

推算时使用干支纪年法。当前年份干支、岁运太过不及、司天在泉、主客气等信息，优先通过以下方式获取：

1. **公元年份 → 干支**：年份-3 除以 60 取余，查甲子表
2. **岁运** = 年干化五运（甲己化土、乙庚化金、丙辛化水、丁壬化木、戊癸化火）
3. **司天在泉** = 年支配对（子午少阴君火/阳明燥金，丑未太阴湿土/太阳寒水，寅申少阳相火/厥阴风木，卯酉阳明燥金/少阴君火，辰戌太阳寒水/太阴湿土，巳亥厥阴风木/少阳相火）
4. **主气** = 按二十四节气分六步（厥阴风木→少阴君火→少阳相火→太阴湿土→阳明燥金→太阳寒水）
5. **客气** = 司天为三之气，依序推定其余五气

实际调用时，如需要完整推算可转 `/tcm-wuyun-liuqi`。
