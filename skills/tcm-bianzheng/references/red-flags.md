# 危重症红旗信号 🚨

> 灵枢台 · tcm-bianzheng 参考文件
> 在辨证全程中，一旦检测到以下任何信号，优先输出红旗警告，暂停常规辨证流程

---

| 信号 | 表现 | 建议 |
|---|---|---|
| 脱证 | 大汗淋漓+四肢厥冷+脉微欲绝 | 🚨 立即就医，考虑参附汤回阳救逆 |
| 闭证 | 神昏+牙关紧闭+两手握固 | 🚨 立即就医 |
| 大出血 | 呕血/便血/崩漏/咯血量多 | 🚨 立即就医 |
| 高热惊厥 | T>39°C+抽搐 | 🚨 立即就医 |
| 真心痛 | 胸痛彻背+手足青至节 | 🚨 立即就医，考虑心梗 |
| 急黄 | 黄疸骤起+神昏+出血 | 🚨 立即就医 |
| 喉痹急症 | 咽喉肿痛+呼吸困难 | 🚨 立即就医，气道紧急 |

---

## 红旗响应模板

出现红旗时输出：

```
⚠️ 注意：你的描述中有 {红旗信号} 的表现。这是一种需要紧急医学处理的危重症。
我建议你立即线下就医或前往急诊，不要仅依赖线上辨证。
如果你确认要暂停辨证，输入 /tcm-save 把当前信息存档，方便后续接续。
```
