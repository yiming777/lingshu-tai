# 体质评估模式

> 灵枢台 · tcm-bianzheng 参考文件

---

## Phase 1B：体质信息采集

说：**「没有病不代表身体没有偏向。告诉我以下情况，我来评估你的素体偏颇倾向。」**

采集九种体质相关核心信息：

```
1. 平时怕冷还是怕热？
2. 容易疲劳吗？疲劳的具体表现？
3. 面色偏红还是偏白偏黄？
4. 舌象特征（如有）？
5. 大便性状和频率？
6. 睡眠质量？
7. 情绪倾向（易怒/易焦虑/平和/容易低落）？
8. 体型（偏胖/偏瘦/适中）？
9. 是否容易上火/生口疮/长痘？
10. 是否容易感冒？
```

---

## Phase 2B：体质判定

按中华中医药学会《中医体质分类与判定》标准，判断主要体质类型（可兼夹）：

| 体质 | 核心特征 |
|---|---|
| 平和质 | 面色红润、精力充沛、耐受寒热 |
| 气虚质 | 气短乏力、语音低弱、易感冒、舌淡胖有齿痕 |
| 阳虚质 | 畏寒肢冷、喜热饮、大便溏、舌淡胖 |
| 阴虚质 | 手足心热、口干、盗汗、舌红少苔、脉细数 |
| 痰湿质 | 形体肥胖、口黏腻、痰多、苔腻、脉滑 |
| 湿热质 | 面油、口苦、苔黄腻、大便黏滞不爽 |
| 血瘀质 | 面色晦暗、舌暗有瘀点、肌肤甲错、脉涩 |
| 气郁质 | 情绪抑郁、胸胁胀痛、善太息、脉弦 |
| 特禀质 | 过敏体质、易发风团/鼻炎/哮喘 |

### 输出格式

```
## 体质评估

### 主要体质
- **{体质类型}**（得分：X/X）

### 兼夹倾向
- {兼夹体质}（如有）

### 调养方向
- 饮食建议：{适合的食物/需要避免的食物}
- 运动建议：{合适的运动类型和强度}
- 情志调护：{情绪调节方向}
- 常用参考方剂：{如玉屏风散/六味地黄丸等，仅供学术参考}

### 运气关照
当前为{年份}年，{运气格局简述}。此格局下{该体质}需要注意{特殊调护建议}。
```
