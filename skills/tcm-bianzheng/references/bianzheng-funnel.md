## 第一层：八纲辨证 + 真假鉴别

**任务**：确定证候的基本属性——表里、寒热、虚实、阴阳。同时排查真假问题。

### 真假鉴别排查清单

**真寒假热排查**：
- 面红如妆 vs 满面通红？
- 口渴喜热饮但不多饮 vs 大渴喜冷饮？
- 脉虽数但按之无力 vs 脉数有力？
- 身热但欲得衣被 vs 恶热喜冷？
→ 有 2 项以上指向真寒假热 → 标注 ⚠️ 真寒假热疑似

**真热假寒排查**：
- 四肢虽冷但胸腹灼热？
- 脉虽沉但按之有力（沉实）？
- 虽恶寒但不欲近衣被？
→ 有 2 项以上指向真热假寒 → 标注 ⚠️ 真热假寒疑似

**真实假虚排查**（大实有羸状）：
- 腹胀满但按之痛剧/拒按 → 实
- 脉虽细但沉取有力 → 实

**真虚假实排查**（至虚有盛候）：
- 腹胀满但时减复如故、按之柔软喜按 → 虚
- 脉虽弦大但按之无力 → 虚

### 输出格式

```
## 第一层：八纲辨证

### 表里定位
- 表证指征：{列举，如恶寒发热、脉浮、头身疼痛等}
- 里证指征：{列举}
- → 判断：{表/里/半表半里/表里同病}

### 寒热定性
- 寒象：{列举}
- 热象：{列举}
- → 判断：{寒/热/寒热错杂}，{如有真假问题则标注 ⚠️}

### 虚实定性
- 虚象：{列举}
- 实象：{列举}
- → 判断：{虚/实/虚实夹杂}，{如有真假问题则标注 ⚠️}

### 阴阳总纲
→ {阴证/阳证/阴阳两虚/阴竭阳脱 ⚠️}
```

## 第二层：脏腑定位

**任务**：根据八纲结果 + 症状定位，锁定病位。

### 五脏定位核心症状

| 脏 | 核心症状 | 舌象指向 | 脉象指向 |
|---|---|---|---|
| 心 | 心悸、怔忡、心烦、失眠、多梦、健忘、口舌生疮 | 舌尖红（心火）、舌淡（心血虚） | 结代（心阳虚/瘀）、数（心火） |
| 肝 | 胁痛、头痛（巅顶/两侧）、眩晕、目赤、口苦、情志抑郁/易怒 | 舌边红（肝火）、舌暗有瘀点（肝郁血瘀） | 弦 |
| 脾 | 脘痞、腹胀、纳差、便溏、乏力、面色萎黄 | 舌淡胖有齿痕（脾虚）、苔腻（脾虚湿困） | 缓弱 |
| 肺 | 咳嗽、气喘、胸闷、鼻塞、咽痛、声音嘶哑、易感冒 | 舌尖红（风热犯肺）、苔白（风寒束肺） | 浮（表证）、细数（肺阴虚） |
| 肾 | 腰膝酸软、夜尿多、耳鸣、脱发、畏寒肢冷/五心烦热 | 舌淡胖（肾阳虚）、舌红少苔（肾阴虚） | 沉弱（肾气虚）、沉迟（肾阳虚）、细数（肾阴虚） |

### 六腑定位核心症状

| 腑 | 核心症状 |
|---|---|
| 胃 | 胃脘痛、恶心呕吐、嗳气、反酸、消谷善饥 |
| 胆 | 口苦、胁痛连肩背、胆怯易惊、黄疸 |
| 大肠 | 便秘/泄泻、腹痛绕脐、里急后重 |
| 小肠 | 脐周隐痛、小便异常、清浊不分 |
| 膀胱 | 尿频、尿急、尿痛、排尿异常 |
| 三焦 | 水液代谢异常、气机升降失常 |

### 输出格式

```
## 第二层：脏腑定位

### 核心病位在 {脏/腑}
- 核心症状支持：{列举 2-3 个最典型的定位症状}
- 舌脉支持：{舌象与脉象如何指向该脏腑}

### 可能涉及的相关脏腑
- {脏腑 A}：{症状}
- {脏腑 B}：{症状}

→ 病位锁定：{主要病位}，兼及{相关脏腑}
```

## 第三层：辨证体系深度分析

根据病位和病性，选择最合适的辨证体系深入分析。**不是所有体系都用——挑最适合的 1-2 个。**

### 3.1 外感病 → 六经 / 卫气营血 / 三焦辨证

**六经辨证**（外感风寒为主）：

| 经 | 核心症 | 脉 | 舌 |
|---|---|---|---|
| 太阳 | 恶寒发热、头项强痛 | 浮 | 苔薄白 |
| 阳明 | 大热、大渴、大汗、脉洪大 / 日晡潮热、腹满便秘 | 洪大/沉实 | 苔黄燥 |
| 少阳 | 寒热往来、胸胁苦满、口苦咽干 | 弦 | 苔薄黄 |
| 太阴 | 腹满而吐、食不下、时腹自痛 | 缓弱 | 苔白腻 |
| 少阴 | 但欲寐、脉微细（寒化）/ 心中烦不得卧（热化） | 微细 | 苔白滑/舌红少苔 |
| 厥阴 | 消渴、气上撞心、心中疼热、饥不欲食 | 弦细 | — |

**卫气营血辨证**（温热病为主）：卫分（发热微恶寒、脉浮数）→ 气分（壮热烦渴、脉洪大）→ 营分（身热夜甚、舌绛、神昏）→ 血分（斑疹、出血、舌深绛）

**三焦辨证**（湿热病为主）：上焦（发热头痛、微恶寒）→ 中焦（身热不扬、脘痞苔腻）→ 下焦（身热颧红、手足心热、舌绛少苔）

### 3.2 内伤病 → 脏腑辨证 + 气血津液辨证

根据第二层锁定的脏腑，进一步辨析：

- 该脏腑的**寒热虚实阴阳**具体类型
- 气血津液的**有余不足**（气虚/气滞/气逆/血虚/血瘀/出血/津亏/痰饮/水湿）
- 脏腑之间的**生克乘侮**关系

### 输出格式

```
## 第三层：{所选辨证体系}深度分析

### 六经/卫气营血/三焦定位（如适用）
- 病程阶段：{经/分/焦}
- 主要依据：{症状+舌脉+病程演变}

### 脏腑辨证深化
- {脏腑}：{证型}——{具体辨析}
- 相关脏腑：{关系}

### 气血津液辨证
- 气：{气虚/气滞/气逆/正常}
- 血：{血虚/血瘀/出血/正常}
- 津液：{津亏/痰湿/水饮/正常}
```

## 第四层：标本缓急

**任务**：区分治标和治本，识别急症需要优先处理的环节。

### 急则治标判断

以下任一标志出现 → 标急于本，先治标：
- 高热不退？
- 剧烈疼痛？
- 呼吸困难？
- 大出血倾向？
- 神昏/抽搐？
- 二便不通？

### 缓则治本判断

- 慢性病程
- 症状不剧烈
- 已经有明确的核心病机
→ 从本论治，或标本兼治

### 输出格式

```
## 第四层：标本缓急

### 当前阶段
→ {急则治标/缓则治本/标本兼治}

### 急标环节（如适用）
- {急证}：{处理方向}

### 治本方向
- 核心病机：{一句概括}
- 治法总则：{如"疏肝健脾，兼清湿热"}
```

## 第五层：兼夹证与转归

**任务**：判断是否有兼夹证候，以及病势的转归方向。

### 兼夹证识别

- 是否有明显的第二个证型并存？（如肝郁化火 + 脾虚湿盛）
- 主证和兼夹证的比例如何？
- 兼夹证是否影响主证的治疗？

### 转归预判

- 当前病势走向：向愈 / 加重 / 传变？
- 可能出现的变化路径（如：肝郁 → 化火 → 伤阴 → 阳亢 → 动风）

### 输出格式

```
## 第五层：兼夹与转归

### 兼夹证
- 主证：{证型名}
- 兼夹：{次证型名}（占比约 {X} 分）

### 转归预判
- 当前阶段：{阶段描述}
- 趋势：→ {可能转归方向}
- 关键监测点：{需要观察的症状或体征变化}
```
