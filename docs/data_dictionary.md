## 一、方剂数据（fangji-db.md）

| 字段 | 中医含义 | 数据类型 | 来源 | 说明 |
|------|---------|:---:|------|------|
| `出处` | 方剂首次记载的经典 | string | 《伤寒论》《金匮要略》《温病条辨》等 | 标注版本（宋本/邓珍本） |
| `组成` | 药物+标准参考剂量（克） | string[] | 《方剂学》教材 + 经典原文 | 剂量为成人一日标准参考量 |
| `功效` | 方剂的治疗作用概述 | string | 经典 + 方剂学教材 | 概括性描述 |
| `主治证型` | 该方对应的主要证型 | string | 经典原文辨证 | 与 zhenghou-db.md 证型一一对应 |
| `煎服法要点` | 特殊煎煮要求 | string | 经典原文 + 临床惯例 | 先煎/后下/包煎/烊化等 |
| `禁忌` | 使用禁忌 | string | 经典+现代药理 | 含人群禁忌+配伍禁忌 |
| `risk_level` | 安全风险等级 | enum(⛔/🟡/🟢) | safety.md 分级标准 | 一级⛔=峻烈/毒性，二级🟡=慎用，三级🟢=常规 |

## 二、中药数据（zhongyao-db.md）

| 字段 | 中医含义 | 数据类型 | 来源 | 说明 |
|------|---------|:---:|------|------|
| `性味` | 四气（寒热温凉）+ 五味（酸苦甘辛咸） | string | 《神农本草经》《中药学》 | 如"辛温""苦寒" |
| `归经` | 药物作用的脏腑经络定位 | string[] | 《中药学》教材 | 如"归心、肝经" |
| `功效` | 药物的治疗作用 | string | 《中药学》教材 | 摘要性描述 |
| `常用剂量` | 成人一日标准用量范围（克） | range | 《中国药典》+ 教材 | 临床实际用量须医师确定 |
| `禁忌` | 使用禁忌和注意事项 | string | 《中国药典》+ safety.md | 含十八反/十九畏/妊娠禁忌 |

## 三、经典条文（jingdian-tiaowen.md）

| 字段 | 中医含义 | 数据类型 | 来源 | 说明 |
|------|---------|:---:|------|------|
| `编号` | 条文在经典中的定位 | string | 原文章节 | 格式如"第 277 条" |
| `原文` | 繁体/原文 | string | 宋本《伤寒论》/邓珍本《金匮》 | 保留原文，不翻译 |
| `白话释义` | 现代白话解释 | string | 教材 + 临床共识 | 学术参考，不同注家解释可能有异 |
| `适用证型` | 条文对应的证型 | string | 辨证分析 | 与 zhenghou-db.md 对应 |
| `临床应用要点` | 临床使用关键 | string | 教材 + 倪海厦注解 | 含方药分析+鉴别要点 |

## 四、证型数据（zhenghou-db.md）

| 字段 | 中医含义 | 数据类型 | 来源 | 说明 |
|------|---------|:---:|------|------|
| `定义` | 证型的精确描述 | string | 《中医诊断学》教材 | 标准学术定义 |
| `主症` | 核心必备症状 | string[] | 教材 + 临床共识 | 缺一则不能定该证 |
| `辨证要点` | 定位要害的关键鉴别点 | string | 教材 + 临床经验 | 最核心的鉴别抓手 |
| `代表方` | 该证型的首选方剂 | string | 经典 + 教材 | 与 fangji-db.md 对应 |
| `病机` | 证型的病机概括 | string | 教材 | 病位+病性+病势 |

## 五、安全数据（safety.md）

| 字段 | 中医含义 | 数据类型 | 来源 | 说明 |
|------|---------|:---:|------|------|
| `risk_level` | 风险等级 | enum(一级/二级/三级) | 临床安全共识 | 一级=必须警告，二级=需提醒，三级=常规 |
| `category` | 风险类别 | enum(方剂/药物) | — | 方剂级风险 vs 单味药风险 |
| `warning_template` | 对应等级的警告文本模板 | string | 自编 | 嵌入式输出 |
| `pregnancy_contraindication` | 妊娠禁忌 | boolean | 《中国药典》 | 禁用=绝对，慎用=权衡后可使用 |
| `eighteen_anti` | 十八反配对 | dict | 《神农本草经》 | 甘草反甘遂/大戟/海藻/芫花 等 |
| `nineteen_afraid` | 十九畏配对 | dict | 《神农本草经》 | 硫黄畏朴硝 等 |
